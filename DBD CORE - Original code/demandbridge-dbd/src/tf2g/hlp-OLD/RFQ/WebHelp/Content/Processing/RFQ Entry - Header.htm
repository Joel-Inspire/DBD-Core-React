RFQ Entry - Header

 

DB Distributor > Request for Quotation System > RFQ Entry - Header (RFQ A1 #1)

 

You will want to fill out the 3 tabs of the header section. Note: Pressing the Next button will not take you to the next tab, it will take you directly to the lines section. You will want to click on each tab to move through the header section.

 

Follow these steps to complete the Header tab:

  1. Select if the quote is for a customer, prospect, or contact. Note: Only quotes for customers can be turned into orders through Order Entry.
  2. Enter in, or lookup and select, the code for the customer, prospect, or contact.
  3. All of their default information will appear. You can edit this as needed.
  4. The order number and order date fields should be left blank. These fields will automatically populate if this quote becomes an order. Continue to the Codes tab.

 

Follow these steps to complete the Codes tab:

  1. Many of the fields on this tab will be defaulted as well. You can edit the defaulted fields as needed.
  2. If this quote has a reference number from the customer/prospect/contact, you can enter it in the reference field.
  3. Optionally, enter in the Action Code of the next action to be taken with this quote. This will vary based upon the stage of quoting you are in. This could be set that you are waiting for the customer, waiting for your vendor(s), that you need to follow-up on production, etc.
  4. If you entered in an action code, enter in the date by which you will need to take or follow-up on that action. This will be used when running the Open RFQ Report.
  5. The print quote field here is related to the printing of the customer quote letter, not the vendor quote. This will usually be defaulted to "yes".
  6. The quote date and quote expiration date fields will default. You can modify these as needed.
  7. Enter in the date that you need the vendor(s) to reply by.
  8. If known, enter in the date that you will need the product by. Continue to the Comments tab.

 

Follow these steps to complete the Comments tab:

  1. Enter in up to 3 comment lines. These will be comments that can print on the Customer Quote Letter. Vendor comments will be created in the vendor section.

 

Press the Next button to continue to the Lines section.
