RFQ Entry - Lines

 

DB Distributor > Request for Quotation System > Processing > RFQ Entry - Lines (RFQ A1 #2)

 

Note:  Line 1 will always be a custom item. Line 2 and beyond can only be message lines or special charges. Since message lines and special charges are self explanatory, this help document will concentrate on adding an item line.

 

Follow these steps to complete the panel:

  1. Select if the line type is new, exact repeat, copy change, spec change, or both change.
  2. Enter in the item code. You also have the option of typing in a new item code to setup an item on the fly. Additional Details
  3. Verify the cost and sell unit of measure that you want to quote the item in. Also verify the quantity contained within those units of measure.
  4. Optionally, click the Specs button to view/modify any of the spec information on this item. This includes the user defined specs, custom item specs, item ply information, standard composition specs, and the item spec comments. Note: Clicking on the Specs button will take you through all five areas just mentioned. You can click on the down arrow portion of the button to receive a drop down list to select one specific area to modify.
  5. At this point you can press the Next or Vend button to continue to the Vendor section of the RFQ.

 

Note: After entering in your vendor information you will be returned to this section. You can then press the Add button to add additional lines (messages or special changes only). If you have no additional lines to add, you can press the Done button. This will return you to the header area where you can save the quote and proceed to the Vendor RFQ Output option to send out the quotes.
