Prospect Name and Address Listing

 

DB Distributor > Request for Quotation System > Prospects > Name/Address List (RFQ P2)

 

The Prospect Name and Address Listing is a report that will list out your basic contact information for your prospects.

 

Follow these steps to run the report:

  1. Select if you want to run the report in order by prospects, or if you would like them grouped by salesperson.
  2. Optionally, enter in a starting and ending range to limit the data that is returned, based upon your selection from step 1.
  3. Review your Output Options.
  4. Press the Print Report button.
