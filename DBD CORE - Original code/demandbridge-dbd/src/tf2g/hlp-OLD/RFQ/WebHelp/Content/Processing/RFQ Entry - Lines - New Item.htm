Item Setup in RFQ

 

If you are setting up a new item during the quote process, the system sets up the item in a temporary item file (QS1). Items do not write out to the permanent item file (FM1) unless the quote is turned into an order. When setting up a new item you will go through all of the standard screens that you normally go through during the custom item setup process. You also have the option of copying information from an existing item when making a new item. This will only pull items that are in the permanent item file. You do not have the option of copying item information from an item that is currently in the temporary item file.
