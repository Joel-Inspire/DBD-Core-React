Vendor Groups

 

DB Distributor > Request for Quotation System > Setup > Supporting Files > Vendor Groups (RFS T1)

 

When entering in your quotes, you can select up to 8 individual vendors to add to the quote, or you can select a vendor group. Having a pre-defined group of vendors can help speed up the quoting process if you know that certain vendors will be used if you are quoting a certain type of product.

 

Follow these steps to complete the panel:

  1. Enter in the group code to setup a new group, or use the lookup to view groups that have already been setup.
  2. Enter in a description for the group.
  3. Enter in up to 8 vendors to be added to the group. For each vendor, you can enter in a specific plant code, if needed. If modifying an existing group, you can add/remove vendors as needed.
  4. Press Save.
