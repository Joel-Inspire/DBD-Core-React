Prospect Setup/Inquiry

 

DB Distributor > Request for Quotation System > Prospects > Setup/Inquiry (RFQ P1)

 

Note: While you can enter in quotes for prospects, these quotes cannot be turned into order like quotes created for customers.

 

Enter in a new prospect code to setup a new prospect, or use the lookup to select an existing prospect to view/modify. Enter in the prospects name, and if needed, a customer code to be billed in their place. You will then need to modify their information across 3 tabs.

 

Follow these steps to complete the Address tab:

  1. Enter in the prospects address, phone, and fax information.
  2. Enter in the prospects purchasing agent and their appropriate greeting.
  3. The prospect status and date setup will be maintained by the system.

 

Follow these steps to complete the Setup tab:

  1. Enter in a category for the prospect.
  2. Optionally, enter in the SIC code for the prospect.
  3. Enter in the sales tax code that would be applied to the prospect.
  4. If applicable, enter in a resale number for the prospect.
  5. Designate if the prospect is tax exempt.
  6. Enter in a terms code for the prospect.
  7. Enter in a credit rating, and a date that rating was assigned.
  8. Optionally, enter in up to 3 mailing label types.

 

Follow these steps to complete the Salesrep tab:

  1. Enter in up to 5 salesreps that are responsible for this account.
  2. Enter in the percent of the available commission that each salesrep should receive. Note: This must add up to 100.
  3. Press the Save button.
