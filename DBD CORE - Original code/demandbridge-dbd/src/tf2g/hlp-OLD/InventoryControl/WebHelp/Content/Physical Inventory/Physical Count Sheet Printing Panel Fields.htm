Physical Count Sheet Printing Panel Fields

 

Field Description
Location The location for the physical inventory.
Salesperson

The salesperson associated with the physical inventory.
Bin Location The bin location.
Customer The customer associated with the physical inventory.
Item Code The item code for the physical inventory.
Sequence The sequence number for the physical inventory.
Do you want to print the computer count Indicates that you want to print the amount the system recognizes as being on-hand.
Page break when bin number changes Indicates that you want to place a page break when the bin number changes.
