Physical Inventory Panel Fields

 

Field Description
Warehouse Code Set Enter a warehouse code set if you would like to freeze inventory for a specific warehouse code set.
Location

Enter a starting warehouse location code if you would like to freeze inventory only for a specific location, or a range of locations.
Through Location Enter an ending warehouse location code if you would like to freeze inventory for a specific range of locations. If you want to freeze inventory for only one warehouse location, enter in the same location as you entered in the Location field.
Cycle Number Enter a cycle number if you have assigned cycle numbers to your inventory items and would like to process only one cycle.
Customer Enter a customer code if you would like to perform a count of only one customer's inventory.
Check Box Select this check box if you want the physical count to be updated with a zero on-hand quantity if no count has been entered for an item.
