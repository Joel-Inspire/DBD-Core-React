Blanket Purchase Orders

 

DB Distributor > Order Processing > Setup > Supporting Files > Optional > Blanket Purchase Orders (OP2 S4)

 

For customers that are using the Order Contracts feature, an enhancement has been added that allows for Blanket Purchase Orders . Blanket PO's can be associated to a specific order contract, and can be applied to an order during Order Entry, after it has been setup here.

 

Follow these steps to complete this panel:

  1. Enter in the Customer that the blanket PO is for. You can also use the 2nd lookup button to view all blanket PO's in existence.
  2. Enter in the blanket PO number.
  3. Enter in the blanket PO date.
  4. Enter in a description for the blanket PO.
  5. Enter in an expiration date for the blanket PO.
  6. Enter in the order contract number that the blanket PO will be tied to.
  7. If only specific operators should have access to this contract, then specify up to 5 operators that will be allowed access.
  8. Enter in a dollar amount for the blanket PO.
  9. The invoice total, total cost, and net sale for orders invoiced to date based upon this blanket PO will be displayed.
  10. Press Save.
