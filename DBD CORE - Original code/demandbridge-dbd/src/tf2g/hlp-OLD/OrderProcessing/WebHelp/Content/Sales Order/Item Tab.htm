Item Tab

 

DB Distributor > Order Processing > Order Processing > Order Entry > Order Entry Defaults > Sales Order Entry > Sales Order Header > Order Line Entry > Item Tab (O/P 01 #2)

 

Use the Item Tab on the Order Line Entry Panel to add a line item to an order. Follow these steps to complete the tab:

 

  1. Enter an item code or select it using the query button. The Modify Item Master? dialog is displayed.
  2. Select Yes to modify the item master on the Custom Items Panel or No to proceed with your order.
  3. Click Next.
  4. The Pricing Tab is displayed.
