Finders Fee

 

DB Distributor > Order Processing > Reports > Order Reports > Finders Fee (OPR R7)

 

Use the Finders Fee report to print what finders fees need to be paid out.

 

Follow these steps to run this report:

  1. Select if you want to run this report by Salesperson or by Vendor.
  2. Optionally, you can mark if you only want to print for paid invoices, and if you want to show/hide the cost and GP columns of the report.
  3. Review the Output Options.
  4. Click Print Report.
  5. After printing the report, you have the option of clearing the information from the finders fee file.
