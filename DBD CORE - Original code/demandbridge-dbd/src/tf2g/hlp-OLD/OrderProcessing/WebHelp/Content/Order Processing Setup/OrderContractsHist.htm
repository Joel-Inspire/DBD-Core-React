Order Contracts History

 

DB Distributor > Order Processing > Setup > Supporting Files > Optional > Order Contracts History (OP2 S3)

 

Use the Order Contracts History panel to view information about orders and invoices that have been associated with your contracts. You can use the lookup button to search for a specific invoice related to a contract, or you can use the navigation buttons in the lower left corner to browse through the records contained in the Order Contracts History file.
