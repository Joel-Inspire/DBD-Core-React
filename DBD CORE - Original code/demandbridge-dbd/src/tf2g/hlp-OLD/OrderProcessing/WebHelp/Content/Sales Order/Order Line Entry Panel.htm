Order Line Entry Panel

 

DB Distributor > Order Processing > Order Processing > Order Entry > Order Entry Defaults > Sales Order Entry > Sales Order Header > Order Line Entry (O/P 01 #2)

 

Use the Order Line Entry Panel to add new lines to an order. This panel contains the following tabs:

 

  1. Line Tab
  2. Item Tab
  3. Pricing Tab
  4. Packing & Shipping Tab
  5. Status Tab
