Ship Via Codes

 

TopForm > Order Processing > Setup > Supporting Files > Recommended > Ship Via Codes (OPS R3)

 

Use the Ship Via Code panel to add your shipping delivery method codes (for example, UPS and FedEx).

 

Follow these steps to complete this panel:

  1. Enter the ship via code that you want to add or modify. You can also use the query button to view existing ship via codes.
  2. Enter a description for the ship via code.
  3. Optionally, you can enter the default tracking website if online tracking is available.
