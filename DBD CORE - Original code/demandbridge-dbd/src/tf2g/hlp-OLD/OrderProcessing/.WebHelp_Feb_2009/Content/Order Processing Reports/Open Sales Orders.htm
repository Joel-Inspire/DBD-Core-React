Open Sales Orders

 

DB Distributor > Order Processing > Reports > Order Reports > Open Sales Orders (OPR R4)

 

Use the Open Sales Order Report to print the orders that you currently have open with your customers. It can show you the total value of the open orders along with estimated gross profits.

 

Follow these steps to run this report:

  1. Select if you want to run the report by Order Number, Salesperson, Division, or Location.
  2. Optionally, you can enter a range of order dates and/or date requested to narrow your search.
  3. Optionally, the remaining questions can be used to narrow your search, and to control what information is printed on the report.
  4. Review the Output Options.
  5. Click Print Report.
