Open Purchase Orders

 

DB Distributor > Order Processing > Purchase Orders > Reports > Open Purchase Orders (P/O RA)

 

Use the Open Purchase Order Report to print any outstanding vendor orders that are delinquent.

 

Follow these steps to run this report:

  1. Select if you want to run the report by Purchase Order Number, Vendor, or Salesperson.
  2. Use the POs to Print field to control what types of POs print on the report.
  3. Optionally, set the "Print P/O in the above range" question to "no" to access the "Print all P/O's which meet any of the following criteria" options, which allows you to further refine your search.
  4. Review the Output Options.
  5. Click Print Report.
