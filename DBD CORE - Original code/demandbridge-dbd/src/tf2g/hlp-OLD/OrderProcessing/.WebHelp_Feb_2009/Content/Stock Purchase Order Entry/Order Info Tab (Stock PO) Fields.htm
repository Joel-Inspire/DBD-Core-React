Order Info Tab Fields

 

Field Description
Print This P/O Indicates that you want to print the order information.
New/Repeat The stock purchase order category. Options include New, Exact Repeat, Copy Change, Spec Change, or Both Changed.
Confirming Indicates that you're confirming a previous order.
Gr. Cutoff Date The Gr. cutoff date. Enter the date or select it using the query button.
Quote The quote for the order.
Date The date of the quote.
Terms The terms for the order.
