Customer Info Tab

 

DB Distributor > Order Processing > Order Processing > Order Entry > Order Entry Defaults > Sales Order Entry > Sales Order Header (O/P 01 #1)

 

Use the Customer Info Tab on the Sales Order Header Panel to define customer information for the order. Follow these steps to complete the tab:

 

  1. Enter the customer number or select it using the query button.
  2. Review the default customer information and optionally enter any changes in the fields.
  3. To repeat a previous order, click Repeat and then click Use this Order on the Repeat Order Entry Panel.
  4. Click Next.
  5. The Order Info Tab is displayed.
