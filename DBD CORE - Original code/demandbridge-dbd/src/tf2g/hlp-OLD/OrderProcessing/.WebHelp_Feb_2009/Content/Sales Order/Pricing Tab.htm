Pricing Tab

 

DB Distributor > Order Processing > Order Processing > Order Entry > Order Entry Defaults > Sales Order Entry > Sales Order Header > Order Line Entry > Pricing Tab (O/P 01 #2)

 

Use the Pricing Tab on the Order Line Entry Panel to add pricing information about the order line.

 

Note: For a Billable Release order, you must choose a warehouse to ship the item from. If you choose a distributor owned warehouse, you will be able to set a sell price. If you choose a customer owned warehouse, you will not be able to set a sell price. You can click the hand icon next to the quantity ordered field to view the available lots if you are using lotted inventory.

For a Release order, you must choose a warehouse to ship the item from, and it must be a warehouse level 9000 or higher. You can click the hand icon next to the quantity ordered field to view the available lots if you are using a lotted inventory. The cost and price amount fields are grayed out because you cannot have charges on this order.

 

Follow these steps to complete the tab: 

  1. Optionally, modify the Unit of Measure and/or Qty Per Unit fields for this order line. The Qty Per Unit field is disabled if you are using a fixed unit of measure such as EACH=1, M=1,000, or DZN=12.
  2. Enter the quantity in the Quantity Ordered field. The system calculates the extension, gross profit, and profit percentage.
  3. Optionally, clear the Fixed Price check box if the customer has not been quoted this price.
  4. Click Next.
  5. The Packing & Shipping Tab is displayed.
