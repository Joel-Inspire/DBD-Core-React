Detail Inquiry

 

DB Distributor > Bank Reconciliation > Utilities > Detail Inquiry (B/R U1)

 

The Bank Rec Detail Inquiry screen is used for viewing records that are currently in the bank rec file, and what their status is. If you are logged in as "super user", you also have the option of modifying the records. When entering into this screen, the system will ask you:

 

Maintain History Detail (N=Active File)?

 

Choosing "yes" means that you will only be viewing historical items.

Choosing "no" means that you will also be viewing active items.

 

From here, you are free to pull up records for viewing as needed, using the Bank Code, Item Type, Reference code, and if needed, the sequence code. You can also use the navigation buttons to move through the records.
