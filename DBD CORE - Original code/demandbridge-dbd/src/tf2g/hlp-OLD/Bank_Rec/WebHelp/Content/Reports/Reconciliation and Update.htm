Reconciliation and Update

 

DB Distributor > Bank Reconciliation > Reports > Reconciliation and Update (B/R P2)

 

Use the Reconciliation Report and Update screen to print a report showing the actions that you have taken in the Bank Statement Data Processing. If all of the information is reported correctly, and you have reconciled, you will then proceed to update the report.

 

Follow these steps to run this report:

  1. Select the bank code that you would like to run the report for.
  2. Select if you would like to print the detail for the cleared items. If not, only outstanding items will be printed.
  3. Review the Output Options.
  4. Click Print Report.
  5. Update the data, if appropriate. If there are any problems with the report, Exit the update and return to the Bank Statement Data Processing to make corrections.
