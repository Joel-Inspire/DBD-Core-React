Import Bank Data

 

DB Distributor > Bank Reconciliation > Utilities > Import Bank Data (B/R U2)

 

Use the Import Bank Data utility to import cleared check data from your bank. Click here for more information on the file to import.

 

Follow these steps to run this report:

  1. Select the bank code that you will import bank data from. Note: A bank reconciliation must be in process for this bank code at the time of importing the data.
  2. The format will be set to "Generic Master". This is the only option available at this time.
  3. Press Proceed.
  4. When the import is finished, you will want to check the log for any issues (as discussed here), and then proceed to the bank rec data entry.
