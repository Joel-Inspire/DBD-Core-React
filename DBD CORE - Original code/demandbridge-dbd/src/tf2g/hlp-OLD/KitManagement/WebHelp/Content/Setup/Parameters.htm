Kit Parameters

 

DB Distributor Optional Modules> Kit Management> Setup > Parameters (KIT S2)

 

The Kit Parameters will need to be setup in order to control the functionality of the module.

 

Please answer the following parameters:

 

1. Always default warehouse code in Kit Component Setup to: ______

Enter a warehouse code. When setting up your Kit components, this is the warehouse that will default to pull the components from.

2. Disable entry of selling price in Kit Production Posting?

If set to "yes", the program will not allow access to the selling price field.

3. Accumulate unit quantity for Inventory Control stats as "K" rather than "U" type?

If set to "yes", the statistics kept in the inventory record for the component item will be kept as "Kit Sales" instead of as "Unit Sales".

4. Enable backorder fill feature from kit production?

If set to "yes", then an existing backorder for a kit item could be filled by producing a kit item trough Production Posting, instead of being filled by PO Receiving only.

5. Default to I/C Base Price for Price during Production Entry?

If set to "yes", then the base price on the lot for the created kit item will default to the base price that is set on the inventory record of the kit item.

 

The press Save to exit and save the changes.
