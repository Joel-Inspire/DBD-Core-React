Web.ec Buyer Setup

The EC Buyer Setup item allows you to specify Web.ec Buyer parameters and controls such as login security, contact information, and catalog controls for each of the approved Web.ec Buyers associated with a Web.ec Customer account.

Definition - Location Based Buyer vs. Purchasing Agent

Buyer Definitions

Web.ec Buyers are assumed to be either tied to a specific ship-to location or doing ordering for any location. These types correspond to our definition of a location based buyer versus a corporate purchasing agent.
