Company Logo images in Web.ec

TopForm WebEC can dynamically display your company logo. The logo is needs to be stored in each customer image directory. The name of the logo file is pulled from the Customer Parameter Setup. If the customer's parameter for company logo is blank, then the logo file name is pulled from the EC system parameters. The following data tags are globally available on all web template pages.

  • company_logo: This tag will return the complete HTML necessary for the display of the logo

  • company_logo_image_source: This tag should be used in the HTML code following the “img src=”. This option allows for you to better control sizing and attributes
