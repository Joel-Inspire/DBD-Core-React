Catalog Code Assignment dialog fields and settings

Clicking the "Catalog Code Assignment" item from the Product Catalog menu launches the Catalog Code Assignment dialog. The Catalog Code Assignment dialog is used to ____________. The following table contains the field descriptions and the setting options and requirements of the Catalog Code Assignment dialog.

Field Name

Purpose / Use

Setting Options and Requirements

Code Type

Select the code type you want to assign, according to the following:

  • C=Customer Category. This setting will allow you to assign the customer category and the catalog to assign in the fields below.

  • B=Buyer Category. This setting will allow you to assign the Buyer category and the catalog to assign. The buyer category is assigned on the second screen of the buyer record.

  • D=Customer/Buyer Category. This setting  will prompt you for a customer code and then a buyer category as described in the fields below.

C / B / D

Buyer Category

("B" or "D" Code Type)

Enter/Select the Buyer category you want to see this catalog. For example, you may want only purchasing agents of all your customers (or a particular customer) to see this catalog. Buyer categories are assigned using the Buyer Setup item.

Web.ec Buyer Category

defined in Buyer Setup

Customer Category

("C" Code Type only)

Enter/Select the Customer Category you want to see this catalog. Customer categories are assigned in the TopForm Accounts Receivable module.

TopForm A/R Customer Category

Customer Code ("D" Code Type only)

Enter/Select the Customer code that has the Buyers you want to see this catalog.
