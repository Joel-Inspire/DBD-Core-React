Rebuilding Web.ec Order Status Sorts

Rebuilding Web.ec order status sort files requires the following user actions.

  1. Launch the Web.ec module.

  2. Select the "Rebuild User Def Order Srt" operation from the System Setup/Utilities menu. The Order Status Sort Rebuild utility starts.

  3. The system will prompt you that it is about to rebuild the order status sort files for the Web.ec system.

  4. When prompted, select "Yes.

  5. Click the button to close the Order Status Sort Rebuild utility dialog.

STOP. You have completed this task.
