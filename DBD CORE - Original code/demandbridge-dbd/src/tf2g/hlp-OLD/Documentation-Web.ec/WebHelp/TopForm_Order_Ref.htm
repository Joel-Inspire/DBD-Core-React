TopForm Order Ref

The "TopForm Order Ref" item is used to return order information to troubleshoot the Variable Print option in Web.ec. This item is only functional if you have the Variable Print module. See the Topform Variable Print documentation for the other requirements to set up the Variable Print option.

 

EC Order Variable Print Info dialog

Selecting the "TopForm Order Ref" item from the System Setup/3rd Party Interface menu launches the EC Order Variable Print Info dialog with all fields empty.
