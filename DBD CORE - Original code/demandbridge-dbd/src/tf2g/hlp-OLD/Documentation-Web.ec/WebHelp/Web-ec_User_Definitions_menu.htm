Web.ec User Defined menu

The Web.ec User Defined menu contains the dialogs used to set the parameters for the Web.ec product catalog and category sort files, system messages, and customer messages. The items the in the User Defined menu include the following:

  • UD Sort Codes

  • UD Sort Element Definition

  • System Messages

  • Customer Specific Messages
