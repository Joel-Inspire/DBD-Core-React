Catalog Category Rename/Delete

This item allows you to copy, rename or delete category entries, catalog categories and all the items associated with them.

 

IMPORTANT NOTE: Before using this function, you should run a catalog report and obtain a printout of the current catalog.

EC Catalog Category Change Utility Dialog

Selecting the "Category Rename/Delete" item from the Product Catalog menu launches the EC Catalog Category Change Utility dialog with all fields empty. This dialog is divided into three tabs.
