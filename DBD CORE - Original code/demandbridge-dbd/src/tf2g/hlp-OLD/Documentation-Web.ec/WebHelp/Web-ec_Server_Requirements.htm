TOPS Web.ec Server Requirements

Web.ec requires a Windows IIS Server or Unix web server environment that supports Perl. The following table specifies the complete hardware and software requirements for a server to accommodate Web.ec.

Requirement

Specification

Platform

Microsoft Windows

Processor

Pentium 100 Mhz (minimum)

Memory

32 MB RAM (64MB or more recommended)

Display

256 Color + VGA Display Adapter (minimum)

Disk Space

115 MB available (minimum

Server Software

Unix Web Server or Windows Internet Information Server (IIS 4.0+ for PC) with Perl-CGI Scripting enabled

Browser

Internet Explorer 4.01 or higher / Netscape 4.6 +
