Web.ec module user interface

The Web.ec module has five menus that contain all the dialogs used to set up and administer the Web.ec software. The Web.ec menus include the following:

  • Setup: This menu is used to set the parameter preferences for the Web.ec E-Commerce system, Customers, Items, Buyers, and order tracking information.

  • User Defined: This menu is used to define the sort codes and system messages used by Web.ec.  

  • Server: This menu is used to setup the Web Server, configure the Server environment, and perform server maintenance tasks.

  • Product Catalog: This menu is used to generate and modify the Web.ec product catalog.

  • System Setup: This menu is used to perform system maintenance tasks such as rebuilding the sort files, mapping product codes, and setting parameters for Variable Print.
