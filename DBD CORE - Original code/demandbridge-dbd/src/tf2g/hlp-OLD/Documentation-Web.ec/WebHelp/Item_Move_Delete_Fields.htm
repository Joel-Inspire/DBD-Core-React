Catalog Item Change Utility dialog fields and settings

Clicking the "Item Move/Delete" item from the Product Catalog menu launches the Catalog Item Change Utility dialog. Using the Catalog Item Change Utility is a three step process. When you commit the settings in this dialog, the system automatically launches the Product Catalog Load Utility and Publish Buyer Catalogs utility to load and publish the new catalog settings. See the  User Instructions for more information.

Catalog Code

Enter/select the code for the source catalog you want to modify.

Web.ec Catalog Code

Item Code Pattern

Enter the Item Code Pattern to use for the items you want to move or delete. Wildcard characters (“?” and “*” ) can be used to identify items for selection. For example, ENV* would find all items starting with ENV and ending with any characters. ENV-??101 would find all items starting with ENV- followed by two characters and ending with 101.

To Category Code

Enter/select the code for the category you want to receive the items. Leave blank if you want to delete the items, and select "Move" in the field below.

Web.ec Category Code

Copy/Move

Determines whether to move or copy the information from the source catalog into the "To" category. Moving to a "Blank" category deletes the items.
