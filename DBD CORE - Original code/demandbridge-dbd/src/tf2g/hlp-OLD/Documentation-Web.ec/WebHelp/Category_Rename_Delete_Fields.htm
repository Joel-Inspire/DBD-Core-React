Catalog Category Change Utility dialog fields and settings

Clicking the "Category Rename/Delete" item from the Product Catalog menu launches the EC Catalog Category Change Utility dialog. Using the EC Catalog Category Change Utility is a three step process. When you commit the settings in this dialog, the system automatically launches the Product Catalog Load Utility and Publish Buyer Catalogs utility to load and publish the new catalog settings. See the  User Instructions for more information.
