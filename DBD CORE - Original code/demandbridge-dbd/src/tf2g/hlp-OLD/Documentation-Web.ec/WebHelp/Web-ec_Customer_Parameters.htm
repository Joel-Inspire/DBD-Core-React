Customer Parameter Setup

The existence of Web.ec parameters for a customer immediately identifies the customer as a Web account. The EC Customer Parameters item allows you to specify customer parameters such as the customer's logo file name, where their images are stored, as well as default operating parameters for each of the approved Web.ec Buyers associated with the account.

EC Customer Parameters Maintenance Dialog

Selecting the "EC Customer Parameters" item from the Setup menu launches the EC Customer Parameters Maintenance dialog with all fields empty.
