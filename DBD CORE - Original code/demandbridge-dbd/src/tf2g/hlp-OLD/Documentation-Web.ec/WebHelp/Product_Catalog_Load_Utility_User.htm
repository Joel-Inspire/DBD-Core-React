Using the Product Catalog Load Utility

To load a TopForm catalog into Web.ec, complete the following actions.

  1. Launch the Web.ec module.

  2. Select the "Catalog Load Utility" item from the Product Catalog menu. The EC Catalog Load Utility dialog appears with all fields empty.

  3. Complete the settings in the dialog according the Dialog Fields and Settings for this topic.

  4. Click the button to commit the field entries.

  5. The system will automatically launch the Publish Buyer Catalogs utility and publish the catalog.

  6. Click the button to close the EC Catalog Load Utility dialog.

STOP. You have completed this task.
