Prod Code to Category Map

Selecting the "Prod Code to Category Map" item from the System Setup/Utilities menu launches the Product Code to Category Mapping utility. This utility is used to map TopForm Product Codes to Web.ec categories. The mapping you set up using this utility automatically associates TopForm Product Codes to Web.ec categories when you run the Product Catalog Load Utility.  

 

See the Dialog Fields and Settings and User Instructions for this topic for more information on using the Product Code to Category Mapping utility.
