Web.ec Catalog Code Based Assignment

The "Catalog Code Based Assignment" item gives you the ability to assign catalogs to a class of customer or buyer, or to buyers of a designated type (location-based or purchasing agent) for a particular customer. For example, if you want to assign a new stock item catalog to all your financial institutions, and you had categorized your customers using the customer category field, you can assign a catalog to the customer category code that you assigned for financial institutions. Web.ec would then use the Catalog ID setting in this dialog for all your on-line financial institutions to see when they login.
