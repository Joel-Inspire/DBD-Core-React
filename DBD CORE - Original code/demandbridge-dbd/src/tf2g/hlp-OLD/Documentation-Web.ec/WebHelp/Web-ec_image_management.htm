Web.ec image management

The first thing you will need to decide is the location of your images. They may reside on your Web Server or your TopForm Server. For recommendations, please contact your TopForm representative. Your main consideration in this area should be both storage capacity of the server and ensuring adequate backup procedures.

Single Image Directory

To configure for a single image directory, simply enter in the image directory path in your TopForm WebEC system parameters. No entry should be made in any Customer Parameters. If you will be supporting form images, you will need to include the customer code as part of the form image in order to avoid duplication of image names if only the form code were to be used.

Separate Image Directories by Client

If you would like to maintain separate image directories by client you will need to enter in the full image path in on the Customer Parameters screen for each client you are supporting on the web. In this situation you may decide to use the form code as the image name since the separate directories by client will ensure no duplication of names even if the same form code is being used for multiple web clients.
