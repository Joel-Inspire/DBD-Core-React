Web Pages and Web Page Templates

With the exception of the login page, all Web.ec pages begin as templates that only identify the format of the page but do not contain any real data. TopForm Web.ec processes these templates and populates them with the appropriate data before displaying them in the user’s browser. See Web.ec transaction processing and Web.ec system components for more information.

Web Server files

The web server holds the login page and the Web.ec client, and may or may not contain image files. Since all other displayed pages contain data from your TopForm system, the originating web page templates are all stored on your TopForm Server.

Naming

Your Web Server will have requirements that the login pages either be called “index.htm” or “default.htm”. Check with you server administrator for details.
