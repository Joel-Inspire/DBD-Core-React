Product Catalog Load Utility

The "Product Catalog Load Utility" item allows you to include or exclude items based on their item selection codes assigned in the Item Parameters Setup. This utility uses the TopForm item selection code and should be used if you expect to be creating many different stock item catalogs that may contain large number of items.

EC Catalog Load Utility Dialog

Selecting the "Load Utility" item from the Product Catalog menu launches the EC Catalog Load Utility dialog with all fields empty. The EC Catalog Load Utility dialog is divided into four tabs.
