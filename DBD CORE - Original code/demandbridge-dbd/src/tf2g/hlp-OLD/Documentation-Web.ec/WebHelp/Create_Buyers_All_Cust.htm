Create Buyers for All Cust

The Create EC Buyer from Customer Master utility is used to set up a Buyer and Customer record for all TopForm A/R customer records that do not have an existing Buyer record on file. The new Buyer record is set up with the TopForm A/R Customer Code as the key, along with any prefix specified in the dialog for the Create EC Buyer from Customer Master utility. When you use this utility, the system uses the Web.ec buyer record 'Default' for the buyer parameters.

 

Note: Each time after you run this utility, TopForm recommends that you also run the Order Status Sort Rebuild utility. See Rebuild User Def Order Srt for more information.
