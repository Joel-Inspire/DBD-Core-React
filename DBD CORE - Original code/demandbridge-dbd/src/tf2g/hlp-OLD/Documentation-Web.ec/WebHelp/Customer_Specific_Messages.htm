Customer Specific Messages

The "Customer Specific Messages" item gives you the capability to give the Buyer(s) for a particular Customer feedback during the order entry process. This feedback is useful in case the Buyer makes an error such as entering an invalid item number, or generates an exception because there is not enough stock for the requested item quantity. You can change the text of the message and use data tags to automatically display to the Buyer information such as the item number they requested and the quantity remaining. The only parameter that is fixed is the message number corresponding to the exception/error.
