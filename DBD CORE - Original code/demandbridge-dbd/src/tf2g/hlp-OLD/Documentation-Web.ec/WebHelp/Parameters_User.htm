Setting Web.ec system parameters

Setting TOPS Parameters requires the following user actions.

  1. Launch the Web.ec module.

  2. Select the "E-Commerce Parameters" item from Setup menu. The E-Commerce Parameters Maintenance dialog appears.

  3. Complete the field entries in the operation dialog according to the Parameters field settings requirements and options.

  4. Click the button to commit the field entries and configure the TOPS parameter settings.

  5. Click the button to close the E-Commerce Parameters Maintenance dialog.

STOP. You have completed this task.
