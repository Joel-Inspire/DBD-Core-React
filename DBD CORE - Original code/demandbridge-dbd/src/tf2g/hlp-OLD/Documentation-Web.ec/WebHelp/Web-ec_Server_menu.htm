Web.ec Server menu

The Web.ec Server menu contains the dialogs used to setup and manage the Web.ec server. The operations you can complete using the items the Server menu include the following:

  • Web.ec Server Setup

  • Server Start/Stop/Status

  • Current Server Sessions

  • Expired Server Sessions
