Customer Specific Messages dialog fields and settings

The following table contains the field descriptions and the setting options and requirements of the Customer Specific Messages dialog.

Field Name

Purpose / Use

Setting Options and Requirements

Customer Code

Enter/Select the Web.ec Customer Code to that you want to apply to this message.

Valid Web.ec Customer Code

EC Program

Enter/Select the pre-defined EC Program code for this message.

Valid EC Program code

Message Number

This field is a counter that is auto-filled by Web.ec, and cannot be modified.

View-Only

Suppress Message Display?

Select this checkbox if you want to suppress the message line display.

Checkbox

Allow Line to be Added to Order?

Select this checkbox if you want to allow the message line to be added to the order.

Checkbox

Order Line Handling/Add Line But Zero Quantity?

Select this checkbox if you want to allow the message line to be added to the order, but not the Quantity.

Checkbox

Order Line Handling/Add Line But Reduce To Max?

Select this checkbox if you want to allow the message line to be added to the order, but want the quantity reduced to the maximum allowable for this customer..

Checkbox

Order Line Handling/Remove Line At Purchase?

Select this checkbox if you want to allow the message line to be added to the order, but removed when the purchase is made.

Checkbox

Order Line Handling/Only Apply Check When Order Review Is Set?

Select this checkbox if you want to allow the message line to be added to the order, only when the order review is set.

Checkbox
