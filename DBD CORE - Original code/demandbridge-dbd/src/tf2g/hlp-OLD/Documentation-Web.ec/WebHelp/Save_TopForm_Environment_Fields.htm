Save TopForm Environment dialog fields and settings

The following table contains the field descriptions and the setting options and requirements of the Save TopForm Environment dialog.

Field Name

Purpose / Use

Setting Options and Requirements

Server Environment Code

Enter the server environment code to use for this TopForm system.

User-Determined (maximum of 8 characters)

Number of File Slots

Enter the total number of file slots in the Unix file system to use for this environment.

Varies by System

Current Environment Information/Company Code

This field displays the Company Code for the current Web.ec environment

View-Only

Current Environment Information/Operator Code

This field displays the Operator Code for the current Web.ec environment

View-Only
