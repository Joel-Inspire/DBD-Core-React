Buyer Type Setup

The "Buyer Type Setup" item is used to create the codes for the Buyer Types that you then assign in the Setup tab of the Buyer Setup parameters. These codes allow you to categorize Buyers for code-based multiple catalog assignment.

Buyer Type Setup dialog
