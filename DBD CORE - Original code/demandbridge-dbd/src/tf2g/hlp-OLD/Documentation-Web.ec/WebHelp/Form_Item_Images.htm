Web.ec Form/Item Images

The decision to place form/item images on your Web.ec pages is a very popular feature that many distributors are looking to offer their web based clients. With TopForm Web.ec you may decide to support form images selectively by buyer. This may be accomplished using the “Web Code” setting on the buyer record which can be set to use HTML templates that have the data tags for the form images.

The administration of you Web.ec site will increase directly with the increase in the number of web based clients for whom you will be supporting images.

Since there is no way in a web page environment to verify if an image is on file. If you decide to support images, you will need to have images for all forms/products placed on the web in order to avoid the populated web pages returning the dreaded “X” image box. You may either store all of your web based images in a single directory or maintain separate directories for each client.

See Form/Item Image Requirements for more information.
