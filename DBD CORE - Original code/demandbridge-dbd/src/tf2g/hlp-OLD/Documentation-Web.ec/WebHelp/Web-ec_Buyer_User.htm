Setting up Web.ec Buyers

To set up a new  Web.ec Buyer or modify the settings for an existing Buyer, you must complete the following actions.

  1. Launch the Web.ec module.

  2. Select the EC Buyer Setup item from Setup menu. The EC Buyer ID Maintenance dialog appears, with all fields empty and only the "Buyer ID" field available for data entry.

  3. Type in the Buyer ID for the new or existing Buyer and press the Enter key. For a new buyer, the fields should now display as ready for data-entry and blank. If the information for an existing Buyer is displayed when you meant to enter a new Buyer ID, this means that an existing Buyer is already using this Buyer ID, and you must select and use a different Buyer ID for the new Buyer.

    Note:     You can click the buttons to browse down or up the list of existing Buyers one-by-one.
     

  4. Customize the new/existing Buyer information according to your requirements and the requirements and options specified in EC Buyer Setup Dialog Fields and Settings.

  5. Click the button to commit the field entries and configure the Web.ec Buyer settings.

  6. Click the button to close the EC Buyer Setup dialog.

STOP. You have completed this task.
