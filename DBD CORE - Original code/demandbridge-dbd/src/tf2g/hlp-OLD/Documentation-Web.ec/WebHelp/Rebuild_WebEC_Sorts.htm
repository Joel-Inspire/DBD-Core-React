Rebuild WebEC Sorts

Selecting the "Rebuild WebEC Sorts" item from the System Setup/Utilities menu launches the Rebuild WebEC Sorts utility. This utility clears and rebuilds sort files for the Web.ec product catalog cross-reference (ECI) and the catalog/category cross-reference (EDI). You need to rebuild these files each time you make changes to your product catalogs or categories.

See User Instructions for the Web.ec interface instructions to use this dialog.
