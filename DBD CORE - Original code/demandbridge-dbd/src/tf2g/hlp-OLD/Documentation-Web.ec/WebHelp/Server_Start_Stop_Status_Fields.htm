Server Status Maintenance dialog fields and settings

The following table contains the field descriptions and the setting options and requirements of the Server Status Maintenance dialog.

Server Status Maintenance

The Server Status Maintenance dialog contains statistics for the following information:

  • Server ID

  • Server Status/Status Time

  • Server Status/Server Start

  • Server Status/Server PID

  • Server Status/Errors

  • Server Status/Current Process

  • Files/Files Done

  • Files/Files to do

  • Orders/Orders Done

  • Orders/Order Lines

  • Logon/Valid Logins

  • Logon/Invalid Logins

  • Logon/Active Sessions
