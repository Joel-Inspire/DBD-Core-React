Creating Buyers for All Web.ec customers

Creating Buyers for All Web.ec customers requires the following user actions.

  1. Launch the Web.ec module.

  2. Select the "Create Buyers for All Cust" operation from the System Setup/Utilities menu. The Create Buyers for All Cust utility starts.

  3. The system will prompt you that it is about to create buyers for all customers in the Web.ec system.

  4. When prompted, select "Yes."

  5. Click the button to close the Create Buyers for All Cust utility dialog.

STOP. You have completed this task.
