UD Sort Codes

The "UD Sort Codes" item gives you the flexibility for defining how Web.ec processes, sorts and displays a number of lists and information sequences. Use this item to view the pre-defined sort codes created by TopForm and to add new sort codes according to your requirements.

 

IMPORTANT NOTE: TopForm pre-defines a number of sort codes for your use (DDL_SHIPTO, EMAILSUB_O, OS_BYDATE, etc.). You can use the UD Sort Element Definition item to define the data and text elements used in these sort codes, but do not modify these codes.
