ID Masters dialog fields and settings

The following table contains the field descriptions and the setting options and requirements of the ID Masters dialog.

Field

Purpose / Use

Setting Options and Requirements

VP Code

This field contains the code that you set up as the Variable Print ID. This ID is used to identify different types of Variable Print items such as business cards and envelopes. If the "Type" field below is set to "S", this field must contain a TopForm shipper code.

Maximum field length is 20 characters

Name

This is the descriptive name you assign to the Variable Print option.

Maximum field length is 20 characters

Field 1 Desc

The description in this field contains the Variable Print order ID that corresponds to Field 160 on Variable Print purchase orders. See the Variable Print documentation for more information.

Maximum field length is 10 characters.                                         

Field 2 Desc

The description in this field contains the name of the proof file

(Example: 55210v.pdf) that corresponds to Field 161 on Variable Print purchase orders. See the Variable Print documentation for more information.

 

VP URL

This field is for the URL to a web page you create where the customer can see info about an order and/or enter a proof number to view the proof.

Example: http://vp.topform.com/500/customer.asp

  • Valid URL to the Customer web page

  • Maximum field length is 60 characters

Proof URL

This is the URL to the directory containing the proofs. If you want to have this URL to appear on your purchase orders for that particular vendor, you must include this field and set up this vendor-specific URL on your ID Masters dialog in the "Third Party Interface" section of the Web.ec Electronic Commerce System (ECS) menu.  

 

Example: http://vp.topform.com/500/proof

  • Valid URL to the proofs directory

  • Maximum field length 60 characters

Vendor's URL

This field is for the URL to the web page you created where the vendor can see info about an order and/or enter a proof number to get the proof. If you want to have this URL to appear on your purchase orders for that particular vendor, you must include this field and set up this vendor-specific URL on your ID Masters dialog in the "Third Party Interface" section of the Web.ec Electronic Commerce System (ECS) menu.

Example: http://vp.topform.com/500/vendor.asp

  • Valid URL to the Vendor web page

  • Maximum field length is 60 characters

 

Type

Select the type of ID Master according to the following:

  • V=Variable Print

  • S=Designates the VP Code as a shipper code for use on the Order Status detail page, to allow linking to a shipper's web site.

V / S
