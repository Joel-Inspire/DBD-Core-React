Rebuild Whse Group Lists

Selecting the "Rebuild Whse Group Lists" item from the System Setup/Utilities menu launches the Update Item Warehouse Groups utility. This utility updates the warehouse group lists to all assigned items. Item warehouse groups are created using the e-Logistics module.

 

Note: This utility is only functional if you also have the e-Logistics module.

 

See User Instructions for the Web.ec interface instructions to use this utility.
