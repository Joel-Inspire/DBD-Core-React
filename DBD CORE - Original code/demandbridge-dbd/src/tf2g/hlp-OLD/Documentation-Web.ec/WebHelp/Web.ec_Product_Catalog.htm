Web.ec Product Catalog overview

Using the Product Catalog Load Utility, Web.ec can take your TopForm product database and automatically "re-categorize" it for publication on the web. You can also create categories based on customer specifications using the Category Setup item, and manually catalogs of stock items as described in Creating Product Catalogs. The Web.ec Product Catalog menu contains the dialogs used to accomplish these tasks, and allows you to customize the categories and how your Web.ec product catalogs appear to Web.ec Buyers.

 

The items in the Product Catalog menu include the following:

  • Category Setup

  • Item Setup

  • Load Utility

  • Publish Buyer Catalogs

  • Category Rename/Delete

  • Item Move/Delete

  • Catalog Code Assignment

  • Catalog Item Move By Description
