Warning: The file "nof_template.htm" cannot be found

This warning appears if the system cannot locate the web page on the TopForm server that was designated in the "Template for template not found field" field of the EC Parameters dialog. To correct this problem, verify that the spelling of the page name in the field matches the filename on the TopForm server. You might also need to contact the TopForm system administrator to check that the page is in the correct directory on the TopForm server.
