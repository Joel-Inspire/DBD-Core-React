Looking up item pricing information

Looking up item price plan and priority information requires the following user actions.

  1. Launch the TOPS module.

  2. Select the "Pricing Lookup" operation from Setup Function menu. The Price Check operation dialog appears.

  3. Click on the button to view/search the entire list of customer codes available in the TopForm Accounts Receivable.

    Note:     You can click the buttons to browse down or up the list of customer codes one-by-one.
     

  4. Enter the United Stationers Item Code for the pricing information you want to view.

  5. Note the item pricing information in the operation dialog "Pricing" tab as described in the Dialog Fields and Settings Help topic.

  6. Click on the "Priorities" tab, and note the item priorities information as described in the Dialog Fields and Settings Help topic.

  7. Click the button to close the Price Check operation dialog.

STOP. You have completed this task.
