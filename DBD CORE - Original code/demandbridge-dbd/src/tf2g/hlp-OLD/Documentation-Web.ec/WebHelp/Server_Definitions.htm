Server Definitions

The following sections define the terms used for the servers that must be administered as part of TopForm Web.ec. See Web.ec system components and Web.ec transaction processing for more information.

Web Server

Server that is connected to the Internet and is hosting your Web.ec site (typically an NT or Linux System).

TopForm Server

Server on which TopForm resides. Typically a UNIX server.

TopForm Web Server

Perl-CGI process running on your TopForm server that processes requests received from your WebEC site.
