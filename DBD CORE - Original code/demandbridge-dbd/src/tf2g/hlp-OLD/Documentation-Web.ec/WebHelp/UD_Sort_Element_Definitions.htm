UD Sort Element Definition

The UD Sort Element Definition item allows you to define the data and/or text elements that make up the sort codes selected and applied using the UD Sort Codes item.

User Defined Sort Element Definition Dialog

Selecting the "UD Sort Element Definition" item from the User Defined menu launches the User Defined Sort Element Definition dialog with all fields empty.
