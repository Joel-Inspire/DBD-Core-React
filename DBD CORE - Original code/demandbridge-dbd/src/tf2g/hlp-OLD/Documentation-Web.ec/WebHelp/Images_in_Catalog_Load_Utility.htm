Images in Catalog Load Utility

Your TopForm WebEC system comes with a catalog loading utility. This utility allows for the loading of forms into the web catalog for a client based on a number of loading criteria. Included in this loading utility is the identification if form image names are to be created as part of the process.

Form Image Format field

The following table specifies the options you have for selecting the format of the form image file names using the Form Image Format field.

Setting

Result

C - Customer Code (raw format) + Form Code

The customer code will contain the two character division number followed by an eight character customer code. There will be no formatting applied to the customer code. If you customer number was  01-00500 then the resulting customer number in a raw format would be 0100500000.

D - Customer Code (formatted) + Form Code

The customer code in this option would be the same as it appears in your TopForm system. If you are divisionalized, it will also contain the hyphen.

F

The image file will be the form code

N

No loading of the form image names
