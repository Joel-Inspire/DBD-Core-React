Web.ec System Setup menu

Each item in the Web.ec System Setup menu contains several embedded items. The items and their embedded functions include the following:

  • Utilities: Rebuild Web.ec sort files and warehouse group lists, create buyers automatically, and map product codes to categories

  • Setup Functions: Save the Web.ec TopForm environment and schedule the Web.ec server tasks

  • 3rd Party Interface: Set up the 3rd party interface for the Variable Print option

  • Supporting File Setup: Set up the supporting files for warehouse group sets, buyer types, ship via map, and companion items
