Creating Catalog Categories

Creating Catalog Categories requires the following user actions.

  1. Launch the Web.ec module.

  2. Select the "Setup" item from the Product Catalog menu. The Product Catalog Category Maintenance dialog appears with all fields empty.

  3. Enter the Category ID or click on the button on the field to view/search the entire list of categories.

    Note:     You can click the buttons to browse down or up the list of categories one-by-one.
     

  4. Create the category according the Dialog Fields and Settings for this topic.

  5. Click the button to commit the field entries.

  6. Click the button to close the Product Catalog Category Maintenance dialog.

  7. Assign the Category to your product catalog(s) and items as required using the Item Setup item.

STOP. You have completed this task.
