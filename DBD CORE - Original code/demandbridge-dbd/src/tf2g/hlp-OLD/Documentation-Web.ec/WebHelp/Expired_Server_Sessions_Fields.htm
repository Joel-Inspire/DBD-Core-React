EC Expired Session Log dialog fields and settings

Clicking the "View Expired Sessions" item from the Server menu launches the EC Expired Session Log dialog. The EC Expired Session Log dialog contains statistics for the following information:

  • Session ID

  • Buyer's ID Code

  • Buyer's Customer Name

  • Buyer's Location Name

  • Session Expired?

  • Session Valid Timeout

  • Login Date Time SQL

  • Last Activity SQL

  • Logoff SQL
