Creating Web.ec System Messages

To create a Web.ec System Message, complete the following actions.

  1. Launch the Web.ec module.

  2. Select the "System Messages" operation from the User Defined menu.  The User Defined System Messages dialog appears with all fields empty.

  3. Enter the appropriate EC Program for your message, or use the buttons to browse the program list one-by-one.

  4. Define the message according to the requirements and options specified in the Dialog Fields and Settings for this topic.

  5. Click the button to commit the field entries and configure the Web.ec system message settings.

  6. Click the button to close the User Defined System Messages dialog.

STOP. You have completed this task.
