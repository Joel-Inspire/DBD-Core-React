EC Variable Print Variable Text dialog fields and settings

The following table contains the field descriptions and the setting options and requirements of the EC Variable Print Variable Text dialog.

Field

Purpose / Use

Setting Options and Requirements

Variable Print ID

Enter the Variable Print ID to assign the variable text. Variable Print IDs are assigned using the ID Masters item.

Maximum field length is 20 characters

Variable Text Code

Enter the numeric code used with the variable text data tag (?tf?3p_text) to indicate the location of the desired text in the text file indicated in the File Name field below.

1-9999

Description

Enter a text description of the Variable text

Maximum field length is 10 characters.                                         

File Name

Enter the file name containing the text returned for the data tag.

[filename].txt
