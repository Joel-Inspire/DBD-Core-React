Web.ec Server Setup

The "Server Setup" item is used to configure the parameters for the server running Web.ec. connectivity can be done either using an internal web server (such as WindowsNT) or an external web server hosted by your ISP. As the configuration requirements of either of these options vary with each different TopForm implementation, direct communication between your technical representative and TopForm technical support is highly recommended.

Note: TopForm strongly recommends that you obtain a secured server certificate for the web server on which Web.ec is running. The page on which the credit card and other sensitive information is being captured can then be done in a secured environment. Please contact your ISP or TopForm if you have any questions.
