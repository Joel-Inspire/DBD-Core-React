Server Maintenance

Server maintenance includes starting stopping, and checking current operating status. Server Maintenance requires the following user actions.

  1. Launch the Web.ec module.

  2. Select the "Server Start/Stop/Status" from the Server menu. The Server Start/Stop/Status dialog appears. When you first enter this dialog you are in the monitoring mode.

  3. The system will display the configured servers and will update the status of the servers every five (5) seconds.  

  4. From the monitoring mode, type “I” to interrupt the monitoring mode and gain access to the maintenance mode.

  5. Complete the required maintenance tasks. See the Server Administration topics and Dialog Fields and Settings for more information. See the following table for setting options.

Option

Action

Comments

Stop a server

Will prompt you for a server id to stop

After entry of a server, the server maintenance program will issue a stop command to the server selected. This will take around five to ten seconds to complete.

Run a server

After entry of a server, the server maintenance program will issue a start command to the server selected. This will take around five to ten seconds to complete.
