Web.ec Buyer login problems

Web.ec Buyer login problems are most often caused by a problem with the Buyer's internal network or Internet Service Provider (ISP). It is therefore recommended that the TopForm administrator or web page designer obtain a name, phone number and email address of a technical representative (IT or web manager) at the buyer's office site, and display it on the web page designated for unsuccessful logins. Then, if a buyer has technical problems, they will call this "Tier 1" person first. This will limit unnecessary usage of the distributor's customer service, salesperson or Web.ec administrator resources. If the buyer's technical representative cannot resolve the problem internally, they will know how to contact the WebEC administrator for assistance.

Warning: The file "relogin.htm" cannot be found  

If the system cannot locate the web page entered in the Template for invalid user/password field, a warning will appear. To correct this problem, check the spelling of the page name or contact the system administrator to check that the page is in the correct directory on the TopForm server.
