EC Tracking Information dialog fields and settings

The EC Tracking Information dialog is a view-only dialog used to track Web.ec sales orders. The EC Tracking Information dialog contains tracking information for the following:

Sales Order (S/O) Number

Don't Bill Order

Sales Order (S/O) Line

Updated Order

Ship To Code

Received

Box Number

Shipped On

Quantity

Estimated Arrival

Actual Weight

Carrier ID/Name

Shipping Weight

Tracking Number

Total Boxes

Pickup Number

Zip Code

Invoiced

Zone

Invoice Number

Total Cost

Packer ID

Lot ID
