Web.ec Initial Setup

The catalog and other data from your TopForm system that Web.ec publishes is not loaded into Web.ec, but is pulled directly from your TopForm database. The initial setup of Web.ec involves setting up the connection between your TopForm Server and your Web Server (see Web.ec system components).

Some setup tasks are required and some are optional based on your particular customer service requirements. The following table lists these tasks in the required order, the Web.ec item and menu or other tool needed to complete the task, and references to the documentation or Help topic that describes the options and requirements for each task.
