Rebuild User Def Order Srt

Selecting the "Rebuild User Def Order Srt" item from the System Setup/Utilities menu launches the Order Status Sort Rebuild utility. This utility clears the User defined Order Status sort file, and rebuilds it from both the open order file and the archived order file. You need to run this utility whenever you modify the sort codes using the UD Sort Codes or UD Sort Element Definition items.

 

See User Instructions for the Web.ec interface instructions to use this dialog.
