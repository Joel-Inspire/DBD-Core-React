Web Development Environment

The establishment of a "development" environment for your Web.ec system gives you the ability, using the same Web.ec interface and controls, to test out new web page templates prior to live implementation. The development system looks at all of the same data as does your live Web.ec configuration to allow you to test out operations using the exact same buyers, customers and catalogs.

 

Note:  Setting up a web Development environment is available but is not standard with your implementation. Please request it from TopForm support.
