Web.ec basic programs

Web.ec "basic" programs are special programs created by TopForm to use the embedded Web.ec page tags to dynamically access and retrieve TopForm data, and generate the output used for the web page.  These programs are launched by specifying the appropriate program name in the Lead-in Parameter basic=[program name]. See Web.ec system components and Web.ec page templates and tags for more information.

The following table describes the purpose and use of the Web.ec "basic" programs. See the document called TopForm Web.ec Web Template Documentation (available on the TopForm web site) for the complete specifications of the Web.ec basic programs.

Basic Program

Generates

Page Function

EC3PC0

Product Catalog page

Used to display the Buyer's designated product catalog. Usually the "landing" page after login.

EC3PC1

Product Catalog Detail page

Used to display details for individual catalog items.

EC3WS1

Order Entry page

Traditional order entry format with the buyer entering their purchase order number, delivery instructions along with items and associated quantities.

EC3OS0

Order Status page

Reports, in a replicated table format, a summarized line of order information.

EC3OS1

Order Status Detail Page

Linked to from the order status (summary) page and is used to display detailed information on the order and the products and delivery information on the order.

EC3PC2

Product Key Word Search page

Used for the buyer to search for an product based on some keyword search.

EC3PC3

Product Key Word Search Results page

Displays the result of the Key Word search. The basic format of the page is designed to be identical to the product catalog page.

EC3RAS

Inventory Report Start Page

Used to specify the criteria that need to be true in order for an item to be reported.
