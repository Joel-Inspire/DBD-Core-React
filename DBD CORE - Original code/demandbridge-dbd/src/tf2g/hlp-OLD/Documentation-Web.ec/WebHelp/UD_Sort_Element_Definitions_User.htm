Defining Sort Code Elements

To define a the elements that make up a Sort Code, complete the following actions.

  1. Launch the Web.ec module.

  2. Select the "UD Sort Element Definition" item from the User Defined menu. The User Defined Sort Element Definition dialog appears with all fields empty.

  3. Select the Customer and the appropriate Sort Code you want to apply to this Sort Code element.

  4. Define the parameters for the Sort Code element according to the requirements and options specified in the Dialog Fields and Settings for this topic.

  5. Click the button to commit the field entries and configure the Web.ec sort code element settings.

  6. Click the button to close the User Defined Sort Code Maintenance dialog.

  7. Repeat steps 1-6 for all the text and/or data elements required for this sort code.

STOP. You have completed this task.
