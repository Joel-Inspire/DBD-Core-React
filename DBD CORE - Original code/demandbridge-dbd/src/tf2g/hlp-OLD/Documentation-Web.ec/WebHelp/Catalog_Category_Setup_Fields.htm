Product Catalog Category Maintenance dialog fields and settings

Clicking the "Category Setup" item from the Product Catalog menu launches the Product Catalog Category Maintenance dialog. The Product Catalog Category Maintenance dialog is used to manually create categories to assign to your product catalogs and items. The following table contains the field descriptions and the setting options and requirements of the Product Catalog Category Maintenance dialog.

Field Name

Purpose / Use

Setting Options and Requirements

Category ID

Enter the name of the product category. See Catalog Category Naming for recommended naming conventions.

User-Determined

Description

Enter the text description you want to appear in the "Categories" drop-down list on the web site.

User-Determined (maximum of 30 characters)

Additional Description

Enter any additional description for this category. The text entered in the Description field may or may not appear on the Product Catalog web page depending on the HTML coding of the pages.

User-Determined (maximum of 60 characters)

Buyer's Location Name

Enter the Location name for location-based buyers.

User-Determined

Secure Codes

Enter the Secure Codes and press the Enter key. Each catalog category can be assigned a maximum of three Secure Codes (two characters each) in this field. In order for a buyer to have this category displayed, they must have a matching two-character Secure Codes in the corresponding field of the Catalog Controls tab in the Buyer Setup.
