Using the Publish Buyer Catalogs utility

Publishing new Buyer catalogs requires the following user actions. Note: This utility automatically runs when you use the Product Catalog Load Utility to load a catalog.

  1. Launch the Web.ec module.

  2. Select the "Publish Buyer Catalogs" item from the Product Catalog menu. The Publish Buyer Catalogs utility dialog appears.

  3. The system will prompt you that it is creating about to publish a new catalog, and gives you the number of items.

  4. When prompted, select "Yes." Wait for processing to complete.

  5. Click the button to close the Publish Buyer Catalogs utility dialog.

STOP. You have completed this task.
