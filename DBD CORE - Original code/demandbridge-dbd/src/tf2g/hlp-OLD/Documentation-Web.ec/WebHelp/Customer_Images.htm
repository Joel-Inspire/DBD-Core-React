Web.ec Customer Images

TopForm Web.ec currently supports the automatic placement of a customer logo on all Web.ec pages. This is accomplished by placing the cust_logo data tag at the location that you desire the logo to be displayed.

Customer Image File Name Specification

You will need to enter in the image file name on the Customer Parameters screen for each of your web based clients. This file name will be appended on to the image path (either from the customer parameter record if separate image directories are desired or from the WebEC system parameters).
