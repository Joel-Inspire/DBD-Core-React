Ship-to Locations

 

DB Distributor > Print Management > Setup > Customer Location (F/M S2) or

DB Distributor > Accounts Receivable > Setup > Customers > Ship to Location Setup (A/R S1 #11)

 

Use the Ship-to Locations screen to add shipping locations for both PM and Non-PM Customers.

 

After entering a customer code, enter in an existing location code to modify, or enter in a new location code to add a new record. Enter in a description for the location, and then you will need to enter in information for the following areas:

  • Address Information
  • Items Management Information
  • Additional Defaults
