Monthly Activity by Warehouse Report

 

DB Distributor > Print Management > Reports > Items > Monthly Activity by Warehouse (FMR RB)

 

The Monthly Activity by Warehouse Report is used to see all activity, by item, for the current PM cycle. Each transaction lists the type of activity (release, receipt, etc.), and the warehouse and/or customer location affected by the transaction. A dollar value is assigned for each transaction. A recap at the end of the report lists total value of Releases, Receipts, Adjustments, and Total Activity. A customer can track the value of inventory held in the distributor's inventory by using this report.

 

Follow these steps to run this report:

  1. This report will run by Item code.
  2. Optionally, enter a range Items, Customers, and/or Locations to narrow your search.
  3. Optionally, the remaining questions can be used to narrow your search, and to control what information is printed on the report.
  4. Review the Output Options.
  5. Click Print Report.
