PM Customer - Item Master Defaults

 

The item master defaults will be used as defaults on new items that are setup for this customer and are marked as FMS items.

Field Description
FMS Flag Y or N. Should new items setup for this customer default to being a PM item?
Restocking Status Should this be based upon a min/max, or based upon a fixed quantity?
Reorder Point in Days Default reorder point in days supply to be set on the main Custom Item Setup.
Standard Supply Warehouse Standard warehouse that their items would be pulled from.
Resupply Minimum in Days Minimum days of supply on-hand before restocking is needed.
Restocking Number of Days Number of days supply to be ordered when restocking.
Resupply Minimum Quantity Minimum quantity on-hand before restocking is needed.
Resupply Restocking Quantity Quantity to be ordered when restocking.
