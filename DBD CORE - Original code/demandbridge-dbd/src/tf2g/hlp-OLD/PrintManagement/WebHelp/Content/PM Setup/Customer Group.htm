Customer Group Setup

 

DB Distributor > Print Management > Setup > Other > Setups and Parameters > Customer Group (FMS S9)

 

Customer Group codes can be used to group similar Print Management customers together for reporting purposes. The PM reports that are available to be printed by Customer Group Code are the Print Management Status Report, Grouping Analysis Report, and the Inventory Valuation Report.

 

Follow these steps to complete this panel:

  1. Enter the customer group code that you want to add or modify. You can also use the query button to view existing codes.
  2. Enter a description for the customer group code.
  3. Enter a group note.
  4. Enter in the main customer code for this group, or select one from the query.
  5. If needed, enter in additional group codes. The customer's information will also be updated to this group.
