Print Management Report Data Update

 

DB Distributor > Print Management > Reports > Customers > Print Management Report Data Update (FMR U1)

 

The Print Management Report Data Update is used for updating your PM data to be available for the PM reports. The PM reports generally will print the information as of the end of the last Cycle Processing, as this update automatically occurs at the end of the PM Cycle. You can also run this update outside of the Cycle Processing to run interim reports for your issue-based PM customers, and count-based PM customers where a mid-cycle count has been performed.

 

Follow these steps to run this report:

  1. Select if you want to run this report by Customer or Group. If you need Interim versions of the report, you will need to click on this option first.
  2. Enter the customer or group that you need to process.
  3. You will then need to confirm if you would like to print interim versions of the PM Reports, and if you would like to update the data for the PM reports.
