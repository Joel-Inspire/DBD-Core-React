3.4 Activate Item

Once the item has been reviewed by the Manager, the item can be activated. This makes it available in the catalog for Buyers to view and order. The Manager clicks on the Activate Item link to be taken to the Activate Item page. The item pricing can be updated at this page and appropriate notes can be entered for this change to the item’s state. Submitting the changes at this time will activate the item. The item status is changed to Item Active. An email notification will be sent to the Designer who created or revised the item if “Yes” is checked against the “Send email notification”.
