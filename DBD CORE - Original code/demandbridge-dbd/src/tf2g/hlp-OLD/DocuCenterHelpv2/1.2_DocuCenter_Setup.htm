1.2 DocuCenter Setup

Separate the users into groups with each group consisting of users requiring similar permissions. Tailor and apply DocuCenter permissions to user groups as appropriate.

Assign administrator users to companies as appropriate.

Assign DocuCenter Purchase Order Managers and DocuCenter Catalog Managers across companies as required.

Assign vendors to appropriate customer companies as required. The vendors will only be able to access the catalogs for their assigned customers only. Other catalogs will be hidden from their view.

Create a set of build sheet product descriptors for the use of Designers in item creation.

For each customer, enter shipping location information.

For each customer, enter billing location information.

For each customer, enter cost center information.

Enter lead time information for each customer as appropriate. The system defaults to a lead time of 96 hours in case there is no lead time set for the customer.
