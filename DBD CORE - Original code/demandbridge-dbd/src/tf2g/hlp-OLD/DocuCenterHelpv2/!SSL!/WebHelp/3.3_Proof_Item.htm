3.3 Proof Item

If “No” is checked for “Proof required” on the Create Item page, the Manager can click on Activate Item. If “Yes” option is checked for “Proof required” on the Create Item page, the Manager clicks on Create Proof Order and is taken to the proof order form. In case this is a brand new item with no existing pricing, then the system redirects the user to the proof pricing page where the Manager is required to setup pricing. The Proof order can be placed after this step.

 

The proof order requests that the manufacturer produce the item using the build information exactly as it is entered by the designer. The shipping information specified by the Designer is used to deliver the proof produced.
