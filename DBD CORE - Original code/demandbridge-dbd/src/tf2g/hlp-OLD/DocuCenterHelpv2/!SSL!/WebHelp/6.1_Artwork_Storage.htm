6.1 Artwork Storage

Artwork is housed on the Incentrix server. Each file has a unique identifier in Incentrix. The artwork file is downloaded by suppliers through items referred to in purchase orders on demand. Any item which is revised still maintains references to its artwork files. Both the old copy and the new copy of a revised item may contain references to the same instance of the artwork file. However, if the row in an item’s buildmap referring to our artwork file is deleted by the designer, this artwork file is going to be orphaned and unused unless it is being used by another item elsewhere; otherwise this artwork file is no longer be in use. It will be taking up expensive space on the server.
