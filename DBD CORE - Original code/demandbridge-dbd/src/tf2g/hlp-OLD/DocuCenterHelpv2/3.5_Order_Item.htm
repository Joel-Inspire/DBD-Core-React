3.5 Order Item

Search for the item on the Search Product Catalog page. Select the Order Item link. Enter the quantity of items you would like to order. Click Submit to add this item to the Shopping Cart. Click on Checkout to go to the Create Order page. Enter the order details including the Shipping details and Billing details. Review the order and submit the order information. An order confirmation page with the system Purchase Order Number will be shown after the purchase order is created successfully.
