General Ledger Journal Update

 

The above message will appear.  Note that the audit report number is this example is 0003.06.  This means that it is the third general journal, and the sixth printing.  (The number preceding the decimal represents the actual journal number and the number following the decimal represents the number of times the report has been printed.)

 

This number will update to the general ledger so that you can trace back the entry.

 

If you press the F10 key or click the “Update data” button, the entries will post to the general ledger.  If you click the F4 key or click the “Exit Update” button, the entries will not update until you again access this program again.  If you have not made any changes since the last printing you have the option to proceed directly to the update.
