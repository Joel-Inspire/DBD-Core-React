Fiscal Year Setup Screen

 

In addition to setting up an entry for the Current Fiscal Year, be sure to set up entries for prior Fiscal Year(s) for which you plan to enter information.

 

Since information is stored internally within the General Ledger System by accounting period within fiscal year, this information cannot be easily changed once postings have been made. Do not Proceed if you are unsure of the information required by this screen. Check with your accounting advisor.

 

Should you have a fiscal year change subsequent to beginning operations, TopForm can assist you in running the Fiscal Year Change utility programs.

 

Fiscal Year Setup Screen Definitions

 

Fiscal Year End

 

Enter the full four digit year, for example, 2004.

 

Number of Accounting Periods

 

Usually this entry is twelve for twelve monthly periods. However, a fiscal year can have from one (1) to thirteen (13) accounting periods.

 

Date that Prior Fiscal Year Ended

 

This date is used to determine the beginning date for the first accounting period.

 

Accounting Period Ending Dates

 

This information will default for standard calendar month periods. If your periods end on some other day, be certain to specify the period ending date for each period. Also, don't forget that February has 29 days in a leap year!
