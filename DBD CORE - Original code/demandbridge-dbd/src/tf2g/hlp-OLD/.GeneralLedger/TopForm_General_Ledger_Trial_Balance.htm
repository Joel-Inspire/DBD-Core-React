TopForm General Ledger Trial Balance

 

A basic rule of double-entry accounting is that for every credit there must be an equal debit amount.  From this concept, one can say that the sum of all debits must equal the sum of all credits in the accounting system.  If debits do not equal credits, then an error has occurred.  The TopForm Trial Balance is a tool for detecting such errors.

 

When viewing the Trial Balance for all accounts, the totals at the bottom of the report should display a beginning and ending balance of zero.  The debit and credit column totals should equal.
