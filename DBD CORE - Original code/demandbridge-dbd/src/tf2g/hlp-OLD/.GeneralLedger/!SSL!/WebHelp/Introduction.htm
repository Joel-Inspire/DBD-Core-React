Prerequisites

 

The General Ledger User's Guide assumes that you have familiarized yourself with the operational aspects of TopForm Accounting. By design, there are no references as to how to enter data into the system.

It is intended that the user be thoroughly familiar with the TopForm System Overview Documentation  before proceeding.
