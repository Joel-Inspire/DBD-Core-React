Parameter - Department Tab

 

Departmental Parameter Setup Screen Definitions

 

Does This Company Have Departments?

 

To enable departmental financial reporting for this company, check this box.  If you do not, the remaining questions within this section will be bypassed.

 

Department Description

 

Examples are Branch, Department, Cost Center and Division. This is for your own information, the description entered into this field will appear as the title of the field in Department Title Setup.

 

 

 

Starting Digit of Departmental Code

 

Financial reporting uses the starting digit in conjunction with the Length of Department Code to determine how this company's chart of accounts specifies department. For example, if the general ledger account number is structured as xxx-xx-xx and the last two digits are to be used as a department code, then you would enter 6 for the Starting Digit of the Department code.  This designation is used both for printing departmental financial statements as well as establishing the parameters for general ledger substitutions.  

 

Length Of Department Code

 

Financial reporting uses this code in conjunction with the starting digit to determine how this company's chart of accounts specifies department.  In the previous example, you would enter 2 for the Length of Department Code. In the panel above, you can see that the department field called Branch is a two character field.

Does This Company Have Sub-Accounts

The following is an example of the use of sub-accounts:

      550-00-00 - Insurance Expense

      550-01-00 – Insurance Expense – General Liability

      550-02-00 – Insurance Expense – Workers Compensation

      550-03-00 – Insurance Expense – Life and Medical

Here 550 is the segment of the account number that defines Insurance. The second segment further classifies transactions as to the type of insurance.

If you check the box, “Does this Company have Sub-Accounts”, you will need to define the segment of the account number that relates to the sub-account. In the above insurance example, you would answer the two questions as follows:

Starting Digit Of The Sub-Account Code?

The answer would be 04. The sub-account starts in the fourth position of the account number.

Length of The Sub-Account Code?

The answer would be 2, as the sub-account has two digits.

Restrict Posting To A Closed Period?

To restrict postings to a closed period, you would need to answer this question, “Y”. However, you may wish to allow certain personnel to post to a closed period. You can accomplish this selective authorization through the “Application Security by Operator” option on System Access and Security Option in TopForm.
