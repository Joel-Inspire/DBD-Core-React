Recurring Journal Entry Setup Panel

 

Recurring Journal Entry is very similar to regular journal entries.   The following are some differences:

 

  • You do not have a reversal option

  • There is no automatic numbering.

  • There are two additional fields to enter the last fiscal year and accounting period to post.  If you were using the recurring journal program to record insurance expense each month with the credit going to prepaid insurance, you would enter the last period that the pre-payment covered.  This prevents you from accidentally making journal entries that no longer are appropriate.

 

You do have the Memo field by line item, as you do in regular journal entries.  We will use the Recurring Journal Entry Program to describe this feature.

 

Click on the Memo Fields by Line box to trigger this feature.  You can still enter up to three line of description as you could on any journal entry.
