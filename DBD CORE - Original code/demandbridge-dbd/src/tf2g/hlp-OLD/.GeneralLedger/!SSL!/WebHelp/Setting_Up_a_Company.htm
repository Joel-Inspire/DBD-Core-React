Setting Up a Company

Procedures - Overview

 

This section describes the steps necessary to properly set up a company for General Ledger Processing.   Although this setup is typically done once, TopForm provides for multi-company accounting.  If you embark upon a new venture, you may need to perform some or all of these procedures for your new company.  Talk to a TopForm support representative about services available to assist you in setting up a new company.

 

To setup a company the following steps should be performed.  More detailed information on each step is covered later in this chapter.  

 

  1. Select the General Ledger option from the Main Menu, then select the Setup Folder (main Setup Folder), then select the System Folder

  2. From the System Folder, select the Parameters Folder, and then the G/L Parameters Option.  

  3. Set the size and format for your general ledger account number.  

  4. Choose the Account Number program option from the main Setup Folder and setup your chart of accounts.  

  5. Return to the Parameters Folder and set the remainder of the parameter options.   (G/L Parameters, Department Parameters, Cash Basis Options, if applicable and Financial Statement Parameters.)

  6. Select the Other Setup Folder and set each of options located in this folder.

 

At this point, all the General Ledger supporting data files will be ready for use.

 

The remaining steps in setting up a company are:

 

  1. Enter the General Ledger Beginning Ba1ances and Verify that the General Ledger balances to your previous accounting system*

  2. Optionally, enter Budget information*

  3. Optionally, enter Prior year information*

 

* See Data Conversion for more information on these setup procedures.

 

Setting Parameters

 

Parameters are used to describe to the General Ledger Programs how to enter and process data for this company. It is actually a way for a company to customize its General Ledger.

 

Parameters, once set, normally should not be changed. Many parameters describe how data is actually stored within the computer and changing their values could confuse the processing programs and corrupt data files.

 

General Ledger Codes

 

In addition to parameters, certain codes are required by your General Ledger System.  They include:

 

  • General Ledger Account Codes

  • Department Titles

  • Source Journal codes

 

In additional, you will need to setup your Fiscal Year and determine the Account Breaks for financial reporting.

 

Data Conversion

 

When setting up a new company, you will need the following:

 

  1. The Period ending dates for the Fiscal (Accounting) Year.

  2. A Chart of Accounts listing the general ledger accounts within each category

  3. A Trial Balance showing ending balances for the last closed accounting period. A Financial Statement will suffice, provided it does not consolidate multiple general ledger accounts onto one line.

  4. A Financial Statement format.

 

The following documents will be needed assuming you want to take immediate advantage of some optional reporting features within the TopForm General Ledger.

 

  1. Budgets by general ledger account for the Current Fiscal Year, if you want to take advantage of Budget versus Actual reporting.

  2. For each general ledger account, the beginning balance and the net change for each accounting period of the Prior Year, if you want to take advantage of current versus prior year reporting.  You may find it more convenient to just let the system accumulate this information from this point forward.
