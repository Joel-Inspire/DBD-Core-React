Historical Balances

If you wish to immediately take advantage of the comparative Financial Statement features, you will need to enter account activity for each period of the current and preceding years(s).

 

To illustrate the first method, lets assume the following:

 

  • Calendar Year Accounting

  • Current accounting date is March 1, 2004

  • Comparative Financial Statements desired for 2003 and 2004

 

Perform the following steps:

 

Using the General Ledger Parameter Setup Program, set the Fiscal Year to 2002, and the Current Accounting Period to 12.

 

Be sure you have set up 2002, 2003 and 2004 in the Fiscal Year Setup program.

 

  1. Using the Journal Entry Program, enter ending balances as of December 31, 2002. Remember, you only need to enter balances for Balance Sheet accounts.

  2. Perform Period End Processing in order to close the year and bring the balances forward as beginning balances for January, 1, 2003

  3. Select the Journal Entry Program and post the net activity for January, 2003.  Do not post the January ending balances but the net change for the period. After printing and updating both the General Journal and the Detail Transaction Postings, select Period End Processing to advance the accounting period. Repeat this step for February through December, 1003.   Period end for December will also perform year-end and advance the system to period 1 of 2004.

  4. Repeat the preceding step for January and February, 2004.

 

Verify that the activity and balances are correct by producing a trial balance and comparing it to the trial balance or financial statement from which you were converting.
