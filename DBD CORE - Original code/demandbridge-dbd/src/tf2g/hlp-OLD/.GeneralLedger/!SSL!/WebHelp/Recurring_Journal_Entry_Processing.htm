Recurring Journal Entry Processing

Recurring Journal Entries can be created for routine monthly journal entries.  Prior to updating, any entries requiring modification can be edited.  This program helps eliminate redundant data entry while at the same time providing a checklist of the entries required for month end.

 

Information Flow for Recurring Journal Entries

 

This section of the documentation outlines the procedure for the Periodic Processing of Recurring Journal Entries.  

 

Details regarding the creation of Recurring Journal Entries will be covered later in this documentation.

 

  1. Choose the Journal Posting Folder from the General Ledger Main Menu

  2. Choose the Recurring Journal and Update Option

  3. Print the Recurring Journal Report and review it for accuracy

  4. If necessary, delete any unnecessary postings and/or edit the amounts.

  5. Reprint the report and choose the update option.

 

The Recurring Journal Update does the following:

 

  • Creates detail postings in the Daily Transaction Register File
