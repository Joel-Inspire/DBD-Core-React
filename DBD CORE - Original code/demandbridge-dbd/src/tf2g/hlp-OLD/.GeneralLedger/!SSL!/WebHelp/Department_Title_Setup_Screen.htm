Department Title Setup Screen

 

This program is used to define the valid departments or branches within the general ledger system.   The length of the Department field was defined in your General Ledger Parameters, Department Panel as two characters.  The department field title of “Branch” was defined there as well.  Set up one department for each of your branch offices or operating divisions.  If you did not check the “Do you have Departments?” box, you will not be able to access this panel.
