G/L Account Break Setup Screen

 

The General Ledger Parameters previously discussed contain the Major Account Breaks that generate totals on the financial statements. This program is used to setup additional subtotals to print on the Financial Statements.  

 

Enter the lowest account number to include in this section of your financial statements and then enter the heading that should print before the section as well as along side the total lines.

 

 

You can use your right query button to see of list of the account breaks you have set up so far (see illustration above).  The breaks shown in this query are fairly typical of account breaks you might want.
