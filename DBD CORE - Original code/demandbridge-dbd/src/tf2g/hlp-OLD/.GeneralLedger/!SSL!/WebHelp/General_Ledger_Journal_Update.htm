General Ledger Journal Update

 

The above message will appear.  Note that the audit report number is this example is 0003.06.  This means that it is the third general journal, and the sixth printing.  (The number preceding the decimal represents the actual journal number and the number following the decimal represents the number of times the report has been printed.)

 

This number will update to the general ledger so that you can trace back the entry to the journal entry report of which it was a part.

 

If you press the F10 key or click the “Update data” button, the entries will post to the general ledger.  If you click the F4 key or click the “Exit Update” button, the entries will not update until you again run this journal report and click the “Update data” button.
