The Detail Transaction Register

The Detail Transaction Register lists every posting that will subsequently be made to the General Ledger.  Each posting is referenced by a Source Journal Code. Based upon the setting in the Source Journal, you have the option to summarize this information before posting to the General Ledger.

 

Information Flow for the Detail Transaction Register

 

Although it is not critical if a day is missed, the following procedure should be performed each day.

 

  1. Chose the Updates Folder from the General Ledger Menu .

  2. Choose the Detail Transaction Register and Update to print the report.

  3. Make certain that the report printed properly and review it for accuracy of the data.  What exactly you should look for in this report will be covered later in this documentation.

  4. If necessary make any additional entries

  5. Reprint the report and give the update command.
