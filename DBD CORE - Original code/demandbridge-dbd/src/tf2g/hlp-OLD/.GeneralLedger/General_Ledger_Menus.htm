General Ledger Menus

The General Ledger Main Menu consists of four main menu folders from which to select the program options.

 

 

Setup Folder

 

This menu contains the program options, that are used when initially setting up a company.  After setup, some of the options will be used occasionally, for example, when a new general ledger account is added to the system.  In addition, this folder contains the Account Balance Option, where you can inquire into the period and year-to-date balance information for each account as well as establish budgets for each account and each period.  We will refer to this folder as the “Main Setup Folder” to distinguish it from some of the other Setup options.

 

 

Notice that within the Setup folder, there is another folder, called System. Within this System Folder, there are four additional folders, System Setup, Parameters, Recurring Journal Posting and Other Setup.  It is within these folders that you will find all the various codes that your General Ledger System will use, as well as the Recurring Journal Posting Setup Option.

 

Reports Folder

 

Use this folder to access your general ledger reports and your chart of account listing.

 

 

 

Updates Folder

 

The Updates Folder contains both the Details Transaction Register and Update programs along with the Period End and Year End Processing Programs.

 

 

Journal Posting Folder

 

The Journal Posting Folder contains the General Journal Entry program as well as the General Journal (report) and Update programs.  Also, the Recurring Journal (report) and update programs are found in this folder.  

Note:  The Recurring Journal data entry program is not located in this folder.  It can be found by following this path, General Ledger/Setup/System/Recurring Journal Posting.
