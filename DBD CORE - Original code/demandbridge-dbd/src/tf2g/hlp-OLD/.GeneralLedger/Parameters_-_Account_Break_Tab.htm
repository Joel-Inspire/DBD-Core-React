Parameters – Account Break Tab

 

 

A new screen will display requesting additional parameter entries for the General Ledger System.  This panel allows you to set the main account breaks

 

Term

Definition/Description

Net Profit Account

Normally, the Retained Earnings Account Number is entered for Post Net Profit at Year End to Account

Asset Account Break

Enter the lowest possible account number to which an asset account will possibly be assigned.

Liability Account Break

Enter the lowest possible account number to which a liability account will possibly be assigned

Equity/Capital Account Break

Enter the lowest possible account number to which an equity or capital account will possibly be assigned.

Revenue Account Break

Enter the lowest possible account number to which a revenue, sales, or income account will possibly be assigned.

Cost Of Sales Account Break

Enter the lowest possible account number to which a cost of sales account will possibly be assigned.

Expense Account Break

Enter the lowest possible account number to which an expense account will possibly be assigned.

Other Income Account Break

Enter the lowest possible account number to which an other income account will possibly be assigned.

Other Expense Account Break

Enter the lowest possible account number to which an other expense account will possibly be assigned.
