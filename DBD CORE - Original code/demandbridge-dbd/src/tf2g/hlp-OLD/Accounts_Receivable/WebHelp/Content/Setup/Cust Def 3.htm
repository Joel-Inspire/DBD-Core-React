Customer Default Data 3

 

Field Description
Freight markup table for invoice You previously were able to enter in a default freight markup table for the customer. You also have the option of using these 3 fields to specify that different freight markups can be applied based upon the 3 invoice types.
Foreign currency to print on invoice If using a foreign currency code, select a foreign currency code description to be printed on the invoice.
Overruns not allowed Ignore this field. Custom for one company.
Sample rebate balance Ignore this field. Custom for one company.
Tax items received into Customer Owned Goods at time of release For Bill as Shipped orders, should the customer pay the tax at the time the goods are put into the warehouse, or should they receive a "tax only" invoice at the time of the items release.
Product Code If you would like to set a default salesperson by a specific product code for this customer, enter in a product code here. If line 001 on the order has this product code, then the salesperson entered in the following field will replace the salesperson that is defaulted on the order header.
Salesperson If you entered a product code into the previous field, enter in the salesperson that should be used as a default for that order.
Substitute digits of warehouse code for packing list printing only ## This method can be used to group all packing lists for this customer.
Immediate print speed bill invoices Designate if they should be printed, prompted for printing, or not immediately printed.
Update credit card orders to summary bill Designate if orders paid by credit card should be updated to the summary bill.
Eliminate posting of sell in lot record for dist owned during PO Recv If you post a product catalog in eCommerce and you want the invoiced price that is displayed to be the actual invoice price, you may want to remove any lot of pricing that would automatically result from PO receiving for this customer. This should be set to "yes" for eCommerce customers using a credit card terms code.
Print lot value for customer owned items in field 46 on packing list This field usually prints the line selling extension. Designate here if you would like to substitute the lot value for customer owned goods.
