Invoice Entry Lines Fields

 

Field Description
Line Type Choose if the line is a Custom Item, Inventory (stock), Customer Inventory, Non-stock, Special Charge, or a Message Line.
Location If an inventory item is on this line, this will show the warehouse it is coming from.
Item Code

The item code for the invoice line. Note: Not used on "non-stock" and "message" lines.
Description Description of the item.
Product Code Product code that the item belongs to.
Unit Unit of measure the item is being billed per.
Qty per UM Qty of eaches within the Unit of measure.
Ordered Quantity of the item that was ordered.
Shipped Quantity of the item that was shipped.
Price Price per the unit of measure.
Price Extension Total sell value for the line.
Cost Cost per the unit of measure.
Cost Extension Total cost value for the line.
Commission Code Commission code that is being used for the invoice line.
Sales Dist Code Sales distribution code that is being used for the invoice line.
Order Type

Shows if this is new, repeat, or other.
GP Adj

Shows if there is a gross profit adjustment on the invoice line.
Comm Amt

Shows the commission amount for the line.
Sales Tax

Shows the sales tax code that is being applied to this line.
Taxable

Shows if the tax status is taxable, non-taxable, or taxed at half rate.
