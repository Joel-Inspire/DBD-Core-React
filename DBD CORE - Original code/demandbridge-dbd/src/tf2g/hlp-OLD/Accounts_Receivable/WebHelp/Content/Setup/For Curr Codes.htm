Foreign Currency Codes

 

DB Distributor > Accounts Receivable > Setup > System > Supporting Files > Foreign Currency Code (ARS T9)

 

This screen allows you to create a Foreign Currency Code and an Exchange Rate, which will be used to translate your standard currency into the foreign currency during the Invoice Entry process.

 

Follow these steps to complete the panel:

  1. Enter in a foreign currency code, or use the lookup to view/edit existing foreign currency codes.
  2. Enter in a description for the foreign currency code.
  3. Optionally, you can enter a three-character description to print on the invoice.
  4. Enter in the exchange rate to be used when figuring the amount to be printed. This field should be updated on a regular basis. However, it can be manually overridden at the time of invoice entry.
  5. Press Save.
