Ship-to Location - Address Information

 

The address information area is used for adding the standard address information for this location code.

Follow these steps to complete this panel:

  1. Enter in a name if there is a specific name associated with this location.
  2. Enter in the physical address information.
  3. Enter in the phone and fax information.
  4. Optionally, enter in an Attention and Greeting for this location.
