Customer Default Data

 

DB Distributor > Accounts Receivable > Setup > Customers > Default Data (A/R S1 #10)

 

The Customer Default Data screens are used for setting up how the customer should be handled in other areas of the software.

 

The customer default data is divided into:

  • Default Data Screen 1
  • Print Management Settings (Optional - FMS customers only)
  • Default Data Screen 2
  • Default Data Screen 3
  • Print Management Fiscal Year (Optional - FMS customers only)
