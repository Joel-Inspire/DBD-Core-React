WordLink Tab

 

Your contact information has pre-populated the WordLink tab.  These entries can be over written, for instance to send the letter to someone other than the main contact or to change the address for the letter.  Each field will be discussed below.

 

Letters Number – Letters are set up in a queue.  This above examples shows that this is the third letter in the queue.

 

Date – This will show as the date on the letter.

 

From – The closing on this letter will include the closing, the name and the title from the data contained in this Salesperson’s file.
 

 

 

In the template drop down box you will be able to see the templates that you have created.  Select the appropriate template and click on the “Word” button.  This will bring up your letter in Microsoft Word and replace the mail merge fields with the data from your Contact.
