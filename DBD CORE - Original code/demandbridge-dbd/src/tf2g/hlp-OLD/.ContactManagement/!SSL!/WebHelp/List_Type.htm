List Type

List Types segregate your contacts into separate lists.  Although you may have some specific applications that would dictate the use of completely separate lists, please note that contacts can be sorted within lists very easily.  Examples of different lists types would be:

 

  • Prospects and Customers

  • Vendors

  • Collections

 

TopForm also uses the List Types to restrict access to prospects and customers by salesperson.  If you wish to take advantage of this feature, you would set up one list type for “All” customers, and then one for each salesperson.

 

To setup List Types, simple choose this first folder from your Supporting Files folder.  Enter in a List Type code up to four characters long.  The system will respond with a dialog box asked you to confirm that you want to set up the new record.  If the List Type already exists, it will display the description and the last number assigned.

 

 

 

Click “Yes,” then fill in the description.  In this case the description would be All Customers.  You will also designate the starting contact code number, by entering the Last Number Assigned.  In this case, we would leave this field set to zero.  The first number assigned would be “1.”

 

If you have a million or more prospects and or customers within any given lists, you would use Alpha Contact Codes.  Enter the last alpha code to assign as 000000.  The first code automatically assigned would be 000001.  After 000009 has been assigned, 00000A, then 00000B, etc.
