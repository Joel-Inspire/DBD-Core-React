Entering Notes

The contact Notes Tab is used to record new notes and to view all previous notes entered for this contact.  To initiate a new note, click on the “Add” button and enter the information.  

 

 

Action Code – Use your query button to chose the appropriate action code or enter the code if know.  The Action description will display.  You can override this description.  In the example above, the operator added their name to the description that displayed.  

 

Notes – Enter your notes.  The notes will wrap when you reach the end of the line, however each note is limited to 164 characters total.  If you need to enter additional notes, enter a new note record.  

 

Follow-up Date – The Follow-up Date, the Follow-up Action Code will be defaulted based upon the entries in the Action Code file.  Please note in the example below, the Action Code “P” has been set to default a Follow-up Action Code of “5” for follow-up in ten days.
