List Profile

 

List Description – If you intend to print a report of the contacts isolated within a given profile, you can enter a description that will print at the top of the report to identify the nature of the profile.

 

                        

List Type – Enter the List Type to query for your profile.  If you need to combine contacts from multiple List Types run the profile a second time.  The qualifying contact form the second list will be appended to the first.

 

Follow-up Date – Only contacts with a follow-up date on or before the date entered here will be added to the profile.

   

                                    

To review more details regarding each of the following codes, refer to the Getting Started section of this documentation.  

 

Follow-up Action Codes – Enter up to sixteen follow up action codes or leave this field blank to include all Follow-up Actions Codes in the profile.  The Follow-up Action Code is found on the Status Tab of your contact File.

 

                     

Note:  If an entry is made in two or more of the following codes, it will assumed that the contact will need to meet both or all conditions to be included in the profile.

  

SIC – This stands for Standard Industry Code.  Only contacts with this SIC code will be included in the profile.  If this field is blank, all SIC codes will be included.

 

Source – Only Contacts with this Source Code will be included in the profile.  If this field is blank, all Source codes will be included.

 

Category – Only Contacts with this Category Code will be included in the profile.  If this field is blank, all Category codes will be included.

 

Minimum Sales – Only Contacts with an Annual Sales equal to or exceeding this value will be included in the profile.  If this field is blank, all contacts will be included.

 

Selection Codes – Selection codes are unique in that your can enter multiple codes that cause a contact to be included, and/or excluded from the profile.  If the contact must meet all the conditions of the Selections codes entered, click the “And” radial button.  If, they need only to meet any one of the Selection codes, click the “Or” radial button.

 

 

 

                    

Range Logic for Profile Creation

 

 

Zip Codes – Ranges of Zip Codes provide the option to create a profile of contacts within certain geographically areas.  You might be calling on accounts and want to review of list of other contacts that might be conveniently located for a sales call.  You may be planning a seminar in a city and want to send out invitations to other contacts within driving distance of the planned event.  

 

Date Entered - range logic provides you with a list of contacts added to your database within a range of dates.  You might want to acknowledge a new account with a “thank you” letter, or follow-up a request for information with an appropriate “second contact” marketing piece.   

 

Last Contact – A sales rep can provide a call report by creating a profile of all calls within a date range.  As a manager, you can view or print a list of all contacts for the week or the month organized by salesperson.

 

Barometers – Barometers of a low of zero to a high of nine can be assigned to grade the quality of the prospect.  Perhaps your sales reps call on contacts with a Barometer of seven to nine and telemarketers call on Barometers three to six.  You may choose to use an occasional direct mail piece for Barometers zero to two.  Of course, these are only examples of how barometer range logic could be used to create profiles for certain actions.       

                     

                                 

Salesperson – You can restrict the profile to only those contacts belonging to a specific salesperson.  

 

Sequence – Use these radial buttons to specify the order in which you would like your profile to be organized.   The choices are: by contact code, by zip code, or alphabetically by lookup code.  You can also specify that they sort first by salesperson, then by contact code, by zip code or by lookup code.

                                   

Remember – with the exception of the Selection Codes where you have the “Or” option, if you have entries in multiple selection criteria, the contact would need to meet all the conditions to be included in the profile.

 

Upon completing your entries, you have the following options.  The “Save” button, initiates the search of your database to create a profile of the contacts that meet the requirements that you have specified.  The “Clear” button clears the “Profile” of any previous selections so that you can start over.  The “Exit” button, exits the profile creation panel.  

 

                           

 

After clicking the “Save” button, you can make additional selections and click the Save button again to add additional contacts to your profile.

 

What can I do with my profile?

 

Profiles can be used to:

  • View the entries one by one and take appropriate action on each

  • Create Letters or mailing labels using WordLink

  • Print Report
