Codes Tab

On the Codes Setup tab, you can enter the Customer code from your Accounts Receivable system, the Salesperson code, the Source code, Category code and SIC code.  Use the query button to look up these codes.   If you have your contacts DUNS number, you can enter it here as well.

 

 

Customer - If there is a Customer code stored in the contact file, you will be able to click on the Customer button.  This will take you to the contact’s customer file where you can view, among other thing, order history, open receivables, inventory, etc.
