Contact Setup

Now you are ready to setup your contacts.  Chose the “Contact” options from the Setup folder.  

 

 

Enter the List Type and then click the Auto button to automatically assign the next contact button.

 

 

You will be prompted to enter an optional follow up comment, a barometer and the customer code.  Normally, you would enter this information each time you initiate some contact with your prospect/customer, etc.  However when setting up contacts for the first time, this dialog box provides you with a convenient method to record this information without navigating to the Notes tab.  

 

You can bypass these fields by clicking “ok”.

 

 

Enter the company name in the Contact Name field along with the other address and contact information.  Note that following entering the street address field, you will be positioned in the Zip Code field, bypassing the City and the State.   Enter a valid Zip Code.  If you have used this Zip Code previously, TopForm will remember the City and the State.  If information regarding this Zip Code has changed, you can edit the file that TopForm stores Zip Code information.  Use the query button to view the Zip code data.  Notice you will be positioned at the Zip code just entered.  Then use the File Folder button within the query to access the Zip Code Setup Program.

      

                               

 

 

Use the Clear button, to clear the fields and enter the zip code information.  Notice that there is a Long City field that is twenty-eight characters in length.  If you enter a city that exceeds the original fourteen-character city field, the City name will be automatically stored here.
