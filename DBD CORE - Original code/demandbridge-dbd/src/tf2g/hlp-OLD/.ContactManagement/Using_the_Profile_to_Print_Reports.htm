Using the Profile to Print Reports

 

There are two Contact Management Reports.  They can be found within the Reports Folder in the Contact Management System.  

 

  • The Contact Follow-up Report

  • The Contact Masterfile Listing

 

Each of the Contact Management Reports can be printed for all or a range of contacts within or across List Types.  

 

 

You can specify whether or not to print notes by checking the Print Notes box.  If printing notes, you can limit the notes to only those subsequent to a certain date, or you can specify a minimum number of notes to print.  

 

 

 

Both reports can also be restricted to only print the contacts contained within your profile.
