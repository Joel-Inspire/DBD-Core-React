WordLink & Letter Writing

WordLink is a TopForm program that integrates the TopForm Contact Management System with Microsoft Word.   Using Word templates you can create letters on a one-up basis or as part of a mailing to a Profile.  Upon selecting the WordLink tab for the first time, the following dialog box will appear.

 

 

This sets the path for your Word templates.  You can accept the defaults by clicking “Save” or override the defaults with your custom settings.
