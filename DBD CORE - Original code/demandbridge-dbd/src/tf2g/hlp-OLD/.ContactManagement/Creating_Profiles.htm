Creating Profiles

 

Profiles provide a means of filtering through all of your contacts to isolate a specific group for mailing or for telephone follow-up.  Profiles can also be used to provide a call report, or for management to review call activity.  

Multiple profiles can exist at the same time as they are user specific, that is, one user can create a profile.  If another user logs in and executes the “Clear” option, and creates a new profile, the first users profile will remain in tact.  There are two ways to access the profiling capabilities within TopForm.  You can select the profile directly from the Contact Management Menu.
