Creating Templates

WordLink is a TopForm program that integrates the TopForm Contact Management System with Microsoft Word.   To use WordLink, for letter writing and envelope and label printing, you will need to set up your templates.

There are three steps in setting up templates:

 

 

Step 1 Create a new folder called TopForm in the following folder on your PC.  This is a one-time setup for each PC.  Save your templates as  *.dot files in this folder.  

 

                          C:\Program Files\Microsoft Office\Templates

 

Step 2 Create your letters in Microsoft Word.

Step 3  Use the mail merge features of Word and WordLink.

  

From “Tools” choose “Mail Merge”,

  

 

 

 

 

After selecting “tmpset.txt,” select “Edit Main Document.”

 

 

In the upper left corner of Microsoft Word you will see the option

 “Insert Merge Field”.  

                              

               The list shown in the illustration to the right represents

                    the fields in Contact Management that are currently

                   available for mail merge.  Select the fields you wish

                                    to have merged into the current document.  

                                   

After you have completed your template you will save

the document in TopForm folder created in Step 1.

 

You are now ready to use WordLink in Contact Management.

 

After creating a profile or selecting a contact, select your WordLink tab.
