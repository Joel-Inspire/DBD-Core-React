Other Utilities

There are other utilities available for performing file maintenance functions.  Most often they are used in conjunction with data conversion to initially setup your Contact Management System.  Talk to your TopForm representative if you need assistance in performing any of the following:

 

  • Copy Customers to Contact Management – This utility will create one contact record for each customer into one specified List Type.  

  • Change Contact List Type – This utility will process the current profile for the user and change the List Type to a new one.  The contact code used for the new List will be based upon the next automatic number for that list.  This utility is typically used to create multiple List Types from the single list created in the first utility.  The first step would be to create the profile, usually for a single salesperson, and then change the List Type for all the customers belong to that salesperson.  Repeat the process until you have created a List Type for each salesperson.  Security by salesperson can then be initiated.  

  • Convert Prospects to Upper and Lower Case – This utility will process the Contact Management file and convert the data from all upper case to mixed upper and lower case.  The following fields will be converted.  The contact name, address lines one and two, the city, contact and greeting.

  • Change Selection Codes – This utility program will process the current profile for the user and for each entry in the profile, change that selection code as specified.  

  • Contact Lookup Maintenance -  This utility program allows you to step through the contact file and change the Contact’s alpha lookup field.  

  • Change / Move Contact Codes – This utility allows you to move a contact from one List to another.  It also provides the option to copy a contact into a second List Type.
