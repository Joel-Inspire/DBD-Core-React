Contact Category Codes

Although they could be used for a variety of purposes, Contact Category codes are generally used to categorize your contacts by industry specification.  Below is a query that displays an example of contact categories.

 

Enter a three character category code and the description to display when this category code is entered.

 

 

 

Please node that the category codes were assigned based upon the alphabetical order of the descriptions.  This or some other logical sequences is recommended for reporting purposes, as certain A/R reports will print in customer category order.  The contact management category code will carry forward to the customer file.
