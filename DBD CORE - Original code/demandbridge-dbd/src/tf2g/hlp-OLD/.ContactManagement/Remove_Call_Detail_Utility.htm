Remove Call Detail Utility

The notes that you enter for your contacts accumulate and are not automatically removed by the system.  Although there usually isn’t any need to remove these notes, TopForm does provide a utility for this purpose.

 

This utility works in much the same way as the List Profile in that you can choose the notes to remove based upon a variety of options.  

 

You should make a backup of your data prior to running this utility.  Following the utility check the data carefully to make certain that you have removed only the notes that were intended.  If you find that you have made an error in the selections, restore your backup.
