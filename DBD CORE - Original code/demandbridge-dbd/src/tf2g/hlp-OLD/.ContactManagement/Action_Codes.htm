Action Codes

In addition to providing a filter for the purpose of sorting prospects, Action Codes, provide a “short hand” method of recording certain actions.  In the example below “P” is short for Phone call.

 

 

Enter the description of the Action Code, and then enter another action code to assign as the follow up code.  You can also enter a default number of days for follow up.
