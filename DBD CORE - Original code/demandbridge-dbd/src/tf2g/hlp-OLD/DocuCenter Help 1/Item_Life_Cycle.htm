Item Life Cycle

Create Item

To create an item, the user must be enabled as a Designer. In the Search Product Catalog page, click on the “Create New Item” button. Enter information required by the form. If a proof is required, click option “Yes” for “Proof required”. Enter proof delivery information. Enter item production specifications for Text and Bindery. This spec information is used by the Manufacturer when the item is produced.

 

The next page presents a form where the user can describe how the item is to be built. Any artwork that is required can be attached at the appropriate step in the build instructions. Each step can only have one artwork file attached to it.

 

If the user is finished with building the item, the item can be submitted to the Manager for review (“Submit Item” button). However, if the user needs to work on the item some more later, the item can be saved for editing later (“Save Item” button). Clicking “Submit Item” makes the item show up in the Manager’s view of the catalog where it can be reviewed. Till such time as the item is not reviewed and processed by the Manager, other users will not be able to see and access the revised item. Clicking “Save Item” only saves the item state and allows this Designer user to access it later for more editing later on.

Review Item

When an Item is submitted to the DocuCenter Manager for review, there is an email notification sent to all DocuCenter managers associated with the current item’s catalog. The Manager can pull up the item for review from the Search Product Catalog page by searching using the item name. Clicking on the item name in the results page shows a summary view of the item. When the item is reviewed, the Manager checks to verify whether the item satisfies the rules and guidelines established for the DocuCenter system.

 

Item Passed Review: If the item does not pass scrutiny, the Manager returns the item to the Designer with an optional email notification using the Activate Item page. The item state is set to Item Saved and submitted. The Manager has the option to send an email notification to the Designer about the item being returned for revision. The optional custom email message can be entered in the Notes text area if the Manager desires.

 

Item Failed Review: If the item passes the Manager’s scrutiny, the Manage Item link allows the Manager to specify the Price and Cost information for the item. Once the item has pricing information associated with it, there are two options at this point.

Proof Item

If “No” is checked for “Proof required” on the Create Item page, the Manager can click on Activate Item. If “Yes” option is checked for “Proof required” on the Create Item page, the Manager clicks on Create Proof Order and is taken to the proof order form. The proof order requests that the manufacturer produce the item using the build information exactly as it is entered by the designer. The shipping information specified by the Designer is used to deliver the proof produced.

Activate Item

Once the item has been reviewed by the Manager, the item can be activated. This makes it available in the catalog for Buyers to view and order. The Manager clicks on the Activate Item link. The item status is changed to Item Active. Submitting the changes at this time will activate the item.

Order Item

Search for the item on the Search Product Catalog page. Select the Order Item link. Enter the quantity of items you would like to order. Click Submit to add this item to the Shopping Cart. Click on Checkout to go to the Create Order page. Enter the order details including the Shipping details and Billing details. Review the order and submit the order information. An order confirmation page with the system Purchase Order Number will be shown after the purchase order is created successfully.

My Catalog

All items which were ordered by a Buyer will be available as long as they remain active in a DocuCenter area called the My Catalog. The My Catalog screen shows all the items from the user’s order history which may still be reordered. Items disappear from the My Catalog when they are deleted by the user or if they are no longer active.
