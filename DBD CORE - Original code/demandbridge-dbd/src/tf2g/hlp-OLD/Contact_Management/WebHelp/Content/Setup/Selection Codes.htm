Selection Codes

 

DB Distributor > Contact Management > Setup > Supporting Files > Selection (T/M S5)

 

Selection codes are used as a means to find certain contacts when creating a list profile. You can assign up to eleven selection codes per contact. You might setup selection codes for tracking anniversaries with customers, to handle collection issues, to track vendor quality, or any other variety of purposes.

 

Follow these steps to complete the panel:

  1. Enter in a 3 character selection code, or use the lookup to view/edit existing selection codes.
  2. Enter in a description for the selection code.
  3. If needed, designate if the selection code should only be made available to certain list types.
  4. Press Save.
