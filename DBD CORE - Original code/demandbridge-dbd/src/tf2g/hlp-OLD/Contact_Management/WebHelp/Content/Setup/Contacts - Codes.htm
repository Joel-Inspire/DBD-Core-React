Contact Setup - Codes Tab

 

DB Distributor > Contact Management > Setup > Contacts - Codes (T/M T1)

 

This tab is used for adding the main required codes for the contact.

 

Follow these steps to complete the tab:

  1. If this contact has a corresponding customer, enter in the customer code. There is a Customer button located to the right that will link you directly to their customer master record.
  2. Enter in the salesperson that is associated with this contact.
  3. Enter in the source code of where the contact originated.
  4. Enter in the category code that best fits the contact.
  5. If needed, select the SIC that best fits the contact.
  6. If needed, enter in a DUNS (Data Universal Numbering System) number for the contact.

 

Continue to the Status tab.
