List Type

 

DB Distributor > Contact Management > Setup > Supporting Files > List Type (T/M S1)

 

When creating your contacts, you will need to place them into one of your pre-defined list types. You could create lists types for customers, prospects, vendors, etc. Another thing to consider is that you have the ability to limit your operators to only be able to view information within certain list types.

 

Follow these steps to complete the panel:

  1. Enter in a 4 character list type code, or use the lookup to view/edit existing list types.
  2. Enter in a description for the list types.
  3. If assigning numeric codes to this list type, enter in the last numeric code that has been assigned. If this is a new list type, this can be set to zero, or a higher starting number if you would prefer. Note: the numeric code has priority over the alpha code if data is entered in both fields. So leave this field blank if you need to use alpha codes.
  4. If assigning alpha codes to this list type, enter in the last alpha code that been assigned. Generally you will be using numeric codes. Alpha codes are usually used if you are manually assigning codes, or if you will need to have more than one million codes assigned within a list.
  5. Press Save.
