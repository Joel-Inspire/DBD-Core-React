Action Codes

 

DB Distributor > Contact Management > Setup > Supporting Files > Action Codes (T/M S4)

 

Action codes serve two purposes related to your contacts. They are a means of recording what actions have occurred with a contact, and what actions should happen in the future with a contact. In the contact record there is a section to record notes. Each note should have an action code associated with it. For example a P to represent a phone call, or an E to represent an e-mail that had been sent. On these notes, you also have the ability to enter in a follow-up action code. For example, if you sent a letter requesting payment, you might enter in a follow-up code to place them on credit hold at a certain date in the future.

 

Follow these steps to complete the panel:

  1. Enter in a one character alpha or numeric code.
  2. Enter in a description for the action code.
  3. If you would like for this action to have a default follow-up action associated with it, enter in the action code for the follow-up. For example, if you left a voicemail, you could enter in the action code that you awaiting a return call.
  4. If the follow-up action should happen at a specific time, enter in the default number of days after the initial action when it should take place. Using our same example, you could set that you will wait 3 days before placing another call after leaving the initial voicemail.
  5. Press Save.
