Bank Reconciliation Parameters

 

DB Distributor > Bank Reconciliation > Setup > Parameters (B/R S2)

 

The Bank Rec Parameters will need to be setup in order to control the functionality of the module.

 

Please answer the following parameters:

 

1. Number of months to retain paid item history: ______

This option affects the B/R Reporting function that reports items that have cleared the bank (paid items) separately from items that have not yet cleared. It would typically be necessary to maintain a paid item history of only one month, as long as reports are maintained.

2. Next automatic adjustment number: ______

This option controls the Automatic Journal Entries Resulting from the Bank Rec and will be numbered beginning with this number. You may want to use a different number range from other journal entries and recurring journal entries to distinguish the Bank Rec entries.

3. Has the bank rec setup utility been run?

This will initially be set to "no". After running the utility, this will be set to "yes". Use the link above for more information about this parameter.

 

Then press Save to exit and save the changes.
