Departmental Chart of Accounts Copy Utility

 

DB Distributor > General Ledger > Setup > System > System Setup > Copy Departmental Accounts (GLS Z2)

 

This utility gives the option of copying all or a part of one division's chart of accounts to another division. This could be used if you have setup accounts for one division that you need to duplicate across other divisions.

 

Follow these steps to complete the panel:

  1. Enter in the division code to be copied from.
  2. Enter in the division code to be copied to.
  3. Select if you would like for account descriptions to be updated if the account already exists.
  4. Select if you would like to copy Balance Sheet accounts as well.
  5. Select if you would like to copy any budget information that might have been setup for these accounts.
  6. Press Proceed to run the utility.
