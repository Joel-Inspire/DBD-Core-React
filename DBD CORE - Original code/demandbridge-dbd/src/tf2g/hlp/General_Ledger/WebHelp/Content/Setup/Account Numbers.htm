GL Account Number Setup

 

DB Distributor > General Ledger > Setup > Account Number (G/L S1)

 

The GL Account Number setup screen is where you will come to setup new general ledger account numbers, or modify/delete existing accounts.

 

Follow these steps to complete the panel:

  1. Enter in a general ledger account code, or select one using the lookup button. Note: the length and format of your general ledger account number is setup in your General Ledger Parameters.
  2. Enter in a description for the account.
  3. Press Save.

 

Note: You can print a list of your accounts by running the Chart of Accounts report.
