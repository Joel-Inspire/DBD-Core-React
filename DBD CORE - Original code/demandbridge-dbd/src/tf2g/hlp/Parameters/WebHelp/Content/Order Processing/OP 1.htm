Order Processing Parameters - Tab 1

 

DB Distributor > Order Processing > Setup > System > System Setup > Parameters (OPS S1)

 

For each parameter listed below, you will see the available options, and if applicable, a recommended setting.

 

Parameter Description
Current Year

Enter in your current fiscal year.
Current Period Enter in your current fiscal period.
Numbering by Division? If set to "yes", the order number will be prefaced with the 2 digit division number. This should be set to "yes" if the AR customer master file is set to divisional.
Finder's fee to affect GP adjustment? If finder’s fees are paid, do you want the amount of the finder’s fee to reduce the Gross profit for commission calculation?
Print plant address on carton label? Will cause the vendor plant address to print on carton labels in carton label printing.
Track promotional statistics? If answered yes, tab 3 will be activated where you will be able to setup your parameters for tracking promotions.
Default number of samples

The number entered here will default on the vendor purchase order.
Quote valid days This defaulted date is normally set to 30 days. Will default on the Customer Quote letter in Request for Quote.
Next quote number Like the order number in OP, or the invoice number in AR, the quote number can be assigned and maintained by the system.
Adjustment factor If printing the plant address on the parameter from above, enter the number 0-6 to adjust the print position.
