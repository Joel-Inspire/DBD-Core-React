Create New Batch Panel

 

DB Distributor > Order Processing > Purchase Orders > Processing > Purchase Order Receiving (P/O P3)

 

Use the Create New Batch Panel to set up a new batch for Purchase Order Receiving. Follow these steps to complete the panel:

 

  1. The new batch number is assigned automatically in the Batch Number field.
  2. Optionally, enter remarks about the new batch in the Remarks field.
  3. Click Save.
