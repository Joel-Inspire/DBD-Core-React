Vendor Info Tab

 

DB Distributor > Order Processing > Purchase Orders > Processing > Purchase Order Entry > Stock Purchase Order > Purchase Order Entry > Purchase Order Setup (P/O P1 #1)

 

Use the Vendor Info Tab on the Purchase Order Setup Panel to define vendor information for the order. Follow these steps to complete the tab:

 

  1. Enter the vendor number or select it using the query button.
  2. Review the default vendor information and optionally enter any changes in the fields.
  3. Select the Order Info Tab.
