Pick Priority

 

DB Distributor > Order Processing > Setup > Supporting Files > Optional > Pick Priority (OP2 S1)

 

Use the Pick Priority setup panel to setup your picking priority options. These will be available for selection in the Warehouse Management System.

 

Follow these steps to complete this panel:

  1. Enter in a Pick Priority code.
  2. Enter in a description for the pick priority code.
  3. Press Save.
