RFQ Parameters

 

DB Distributor > Request for Quotation System > Setup > System Setup > Parameters (RFS S1)

 

Answer the following parameters:

 

1. During item copy, clear all statistics and previous order data?

When copying an item, would you like to clear any statistics and order data related to the original item? This will normally be set to "yes".

2. Ask to immediately print quotes to vendors?

After entering in a quote, would you like the system to automatically prompt you to print/fax/e-mail the quote? We would recommend setting this to "yes".

3. Set vendor print defaults to "P" on auto created vendor groups?

When pulling in a vendor group on a quote, do you want to set all of them to print by default? This would normally be set to "yes".

4. Quote Manager e-mail: _______________________

If you have an internal quote manager that should be alerted on all RFQs entered, you can enter their e-mail address here.
