Customer Default Data 2

 

Field Description
Minimum Percentage markup for all lines If desired, enter in a minimum percentage to have lines marked up.
Require entry of "RC number" in Order Entry Set this to "Y" to make the "Remote Cost" field in Order Entry a mandatory field.
Require entry of "Ordered by" in Order Entry Set this to "Y" to make the "Ordered by" field in Order Entry a mandatory field.
Foreign Currency Code If this customer needs to be invoiced in another currency, enter the foreign currency code here.
CC Plus, Alt terms code if invalid credit card If a credit card is declined, enter in the terms code that should be used as a backup.
Job card override in Invoice Printing If you are using electronic forms for printing, and you need special instructions for this customers invoices, you can list the instructions here. When printing invoices, you will need to make sure that you allow the job card override.
Picking Priority If this customers items need to be shipped out in a timely manner, you can assign a picking priority to make sure that their items are picked and shipped sooner than that of other customers.
Send inventory quantities to external system via XML To have this customer's inventory balances sent to Four51 via XML, set this to "yes".
If no whse specified in incoming XML order, default whse to use If an order comes in from an outside system that does not specify a warehouse, you can specify a default warehouse for regular orders and purchase orders.
Retain sell price during BO Fill Should the sell price be kept during the backorder fill report?
Require entry of "Contract Number" in Order Entry Set this to "Y" to make the "Contract Number" field in Order Entry a mandatory field.
Merchant Account If this customer has a standard merchant account, enter it here.
