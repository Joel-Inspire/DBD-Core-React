Vendor Plant Setup

 

DB Distributor > Accounts Payable > Setup > Vendors > Plant Setup (A/P S1 #8)

 

Similar to your customers having multiple ship-to locations, your vendors might have multiple plant locations that you order from. Use this panel to setup any location addresses for your vendors.

 

Follow these steps to complete the panel: 

  1. Enter in a new plant code to setup a new vendor plant location, or use the lookup to view existing records.
  2. Enter in a name for this location.
  3. Enter in all of the standard address information.
  4. Enter in the remaining phone/fax and contact information.
  5. Press Save.
