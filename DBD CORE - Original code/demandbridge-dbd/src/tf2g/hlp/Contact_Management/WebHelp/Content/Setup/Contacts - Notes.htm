Contact Setup - Notes Tab

 

DB Distributor > Contact Management > Setup > Contacts - Notes (T/M T1)

 

This tab is used for taking notes on the various interactions that you have with the contact. It is also used for reviewing historical notes.

 

Follow these steps to add a new note:

  1. Press the Add button.
  2. Enter in an action code that best fits the type of action that was taken.
  3. Enter in any notes related to this action.
  4. The follow-up date and follow-up action code fields will default if there was one associated with the action type you are entering. These fields can be modified if needed. Also, to have this follow-up action placed into your Microsoft Outlook calendar, click on the Outlook button located to the right of the notes field.
  5. Optionally, you can also reset the barometer field if needed.
  6. Press the Save button to save the note.

 

Follow these steps to view historical notes:

  1. Press the History button.
  2. A listing of all notes with this contact will be displayed, with the most recent notes appearing at the top.
  3. Click on any note and press the Edit Note button to make modifications to the note.
  4. Click on any note and press the Delete Note button to delete the note.
