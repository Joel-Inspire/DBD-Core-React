List Profile

 

DB Distributor > Contact Management > Setup > List Profile (T/M T2)

 

The List Profile screen is used to search through your contact records based upon specific criteria entered here, and build a list of the findings. The list profile is specific to each user on your system. If you have previously created a profile, you will have the option of clearing it when you re-enter this screen. If you choose not to clear it, and run the profile again, you will add to the records already existing.

 

Follow these steps to create a list profile:

 

  1. If needed, enter in a list description to describe the list you are going to create. This will print as the title if you choose to run the Contact Follow-up Report or the Contact Masterfile Listing report with the list profile option turned on.
  2. Enter the List Type to query for your profile. If you need to combine contacts from multiple List Types, you can run the profile a second time. The qualifying contacts from the second list will be appended to the first.
  3. If you only want contacts considered that have a follow-up date before a certain date, enter that date in.
  4. If you only want contacts with certain follow-up codes to be considered, enter in up to 16 follow-up codes side by side.
  5. If you want to include contacts with a specific SIC, Source, and/or Category, enter in those fields as needed.
  6. To include only contacts with a minimum annual sales number, enter in the minimum amount.
  7. Optionally, enter in a range of zip codes to narrow the search.
  8. Optionally, enter in a range of original dates entered to narrow the search.
  9. Optionally, enter in a range of last contact dates to narrow the search.
  10. Optionally, enter in a barometer range to narrow the search.
  11. Optionally, enter in up to 12 selection codes to include in the search. You can specify that they have to contain all selection codes by using the "And" button, or that they only need to contain one of the selection codes by using the "Or" button.
  12. Optionally, enter in up to 12 selection codes to exclude from the search. You can specify that they have to contain all selection codes by using the "And" button, or that they only need to contain one of the selection codes by using the "Or" button.
  13. If you only want to include contacts with a certain salesperson, enter in that salesperson code.
  14. You can use the Sequence field to indicate which order you would like the profile organized in.
  15. Optionally, you can enter in a category set and/or state set to further narrow the search.
  16. Press Save to have it save the profile.

 

Once the profile has been saved, you can use it in the following ways:

  • Create letters or labels for these contacts using Wordlink
  • Run the Contact Follow-up Report and only display these contacts
  • Run the Contact Masterfile Listing report and only display these contacts
  • Navigate through your contact records using only these contacts
