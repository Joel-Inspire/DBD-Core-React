Contact Setup - Wordlink Tab

 

DB Distributor > Contact Management > Setup > Contacts - Wordlink (T/M T1)

 

This tab is used for passing information into Microsoft Word, Excel, or Outlook. All of the information on this tab will default based upon the information that was entered on the main tab. Any changes that you make to the data on this tab will just be temporary changes, they will not be saved back to the main tab.

 

Follow these steps to complete the tab:

  1. Modify any of the data fields as needed.
  2. Select a template from the drop-down list.
  3. If sending an e-mail, enter in a subject line to be used.

From here you will have the following options:

  • Press the Excel button to send the data from this record into an Excel spreadsheet.
  • Press the Word button to send the information from this record into a Word document using the template selected in step 2.
  • Press the E-mail button to have an e-mail sent to the contact, with a Word document attachment based upon the template selected from step 2. Before doing this, you will want to press the Create Message button in order to type in the body of text that would be sent in the e-mail.

 

Return to Contact Setup

 

Note: If you have the Profile turned on, you can press the Clear button while on this tab to clear any information, then select a template, and then press the Word button to create a letter for each contact stored within the profile.
