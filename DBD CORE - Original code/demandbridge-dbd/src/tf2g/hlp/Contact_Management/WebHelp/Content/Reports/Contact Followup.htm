Contact Follow-up

 

DB Distributor > Contact Management > Reports > Contact Follow-up (T/M R1)

 

The Contact Follow-up Report is used to provide you a listing of your contacts, and when your last contact with them was. You have the option of printing their notes as well.

 

Follow these steps to run this report:

  1. Select if you would like to run this report sorted by the list type and contact code, or have it sorted by the list type and lookup code.
  2. Optionally, enter in a range to narrow your search. The criteria used to narrow your report will determined by the sequence you chose to run the report by in step 1.
  3. Select if you would like to print the contact notes on the report.
  4. If printing notes, you have the option of only printing notes that were added after a specific date.
  5. Select if you would like to have a minimum number of notes print. This will print older notes if enough were not found within the date range you specified.
  6. Select if you would like to limit the report to only contacts that are currently included in your list profile.
  7. Review the Output Options.
  8. Click Print Report.
