Contacts

 

DB Distributor > Contact Management > Setup > Contacts (T/M T1)

 

The Contact Setup panel is used to setup new contacts, and it is also used for displaying and/or changing information on existing contacts.

 

Follow these steps to complete this panel:

  1. Select the list type that the contact should belong to.
  2. Enter in a new contact code for the list type, or press the Auto button to assign the next available contact code for this list type.
  3. If this is a new contact, enter in the contact name.

 

You will then need to supply information that is spread across 5 tabs. Select from the following tab options:

 

  • Main - Includes primary contact information.
  • Codes - Includes main codes to categorize and identify the contact.
  • Status - Includes current status information about the contact.
  • Notes - Allows you to add new notes to the contact, and review historical notes.
  • Wordlink - Allows you to pass information into Word, Excel, or Outlook .

 

Note: The Profile button can be used to control which records you are able to view from this screen. Clicking on this button brings up a panel with 4 more buttons on it. From here, you can turn the Profile feature On or Off, by using the corresponding buttons. When the profile is turned on, you can use the arrow buttons at the bottom of the screen to navigate through the records that are part of the profile. Using the lookup button on the contact field will still result in all contacts being viewed. If you press the Maintain button, this will link you to the List Profile screen, where you can modify the criteria used to create the profile.
