Rebuild eCommerce Sorts

 

DB Distributor Toolkits > eCommerce > Setup > System > Utilities > Rebuild eCommerce Sorts (ECS U1)

 

Automatic Invoice Selection is used to let the system automatically select vendor invoices to be paid, based upon ranges that you specify. You can enter information into different fields to control what invoices will be selected. Here are the fields available and how they control the resulting selection.

 

Field Description
Take Available Discounts Expiring After Enter in a date to use for considering whether or not discounts should be taken. Note: This first field is not used for controlling what invoices should be selected, but to determine if discounts should be taken on the invoices that are selected.
Add Only Invoices with Discounts Expiring From

If needed, enter in a discount date range to select invoices that have an applicable discount within a given time period. This would be used, for example, if you wanted to select only invoices that offered a discount within the next week.
Add Only Invoices Due on or Before If needed, enter in a date here to control which invoices will be considered for selection based upon their due date.
Add Only Category Code If needed, enter in a vendor category code to select only invoices that have a certain category code assigned to them.
Add Only Bank Account If needed, enter in a bank account code to select only invoices that would print for a specific bank code.
Only Add Priority Codes Greater Than If needed, enter in a priority code A through Z. Only priority codes higher will be considered for selection.
Only Add Payment Selection Code If needed, enter in a payment selection code. Only invoices with this payment selection will be considered for selection.
Enter Starting and Ending Vendors to Select If you want to pay vendors within a range based upon their vendor code, enter in a starting and ending vendor code to specify that range.

After making your selection(s), click on the Proceed button to have the system read through the Open Invoice file and select invoices to be paid. You will then continue to the Suggested Payment Report to view a listing of invoices that were selected to be paid. Also, you have the option of going to the Specific Invoice Selection screen to manually add or remove invoices from those that have been selected.
