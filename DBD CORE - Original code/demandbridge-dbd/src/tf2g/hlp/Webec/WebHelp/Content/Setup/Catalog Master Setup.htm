Catalog Master Setup

 

DB Distributor Toolkits > eCommerce > Setup > System > System Setup > Catalog Master Setup (ECS S3)

 

The Catalog Master Setup screen is used for creating new catalogs, or modifying the properties of existing catalogs.

 

Follow these steps to complete the panel: 

  1. Enter in a catalog ID. You can also use the lookup button to view a listing of exiting catalogs. Note: a commonly used method for naming catalogs is to use the customer name as the catalog ID.
  2. Enter in a description for the catalog.
  3. Select if the catalog will be used for selling custom items or stock items.
  4. Enter in a description to be used for the catalog in drop down lists. This will be displayed on the product catalog page if there are multiple catalogs used by a buyer.
  5. Select if you would like for this catalog to be restricted from reporting.
  6. Select if you would like for the catalog items to be sorted by item description, as opposed to the item code.
  7. Select if you would like this catalog to be restricted from showing in drop down lists on the product catalog page.
