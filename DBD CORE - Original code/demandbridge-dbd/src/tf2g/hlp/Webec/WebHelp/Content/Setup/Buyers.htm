Buyer Setup

 

DB Distributor Toolkits > eCommerce > Setup > (E/C S4)

 

Before your customers can begin entering orders online, they will need to have buyer records setup here. You can click on the lookup button to see a listing of existing buyer codes setup in your DB Distributor system. Select a buyer code to setup or modify. If setting up a new buyer, enter in the buyer name.

 

Note: the buyer ID's are global, not customer specific. So, for example, if you have a John Smith at two different customers, you cannot give them both the buyer ID of jsmith. The buyer name can be displayed on the web page templates using the ?tf?buyer_name? tag.

 

You will then need to fill out the buyer information, which is spread across the following seven tabs:

  1. Setup
  2. E-mail
  3. Order Status
  4. Catalog
  5. Order Entry
  6. Credit Card
  7. Other
