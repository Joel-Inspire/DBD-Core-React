DB Distributor Toolkits > eCommerce > Setup > System > 3rd Party Interface >ID Variable Text Ref (ECS T2)

 

In the Accounts Payable parameters, you can indicate if you process 1099's. If you do, this panel will be activated. Most of the information on this panel is automatically updated through your daily processing.

 

Follow these steps to complete the panel: 

  1. Enter in the vendors tax id number.
  2. Enter in the 1099 type to be printed.
  3. Press Update.
