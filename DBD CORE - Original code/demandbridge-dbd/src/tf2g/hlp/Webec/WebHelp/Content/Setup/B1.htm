Buyer Setup - Tab 1 - Setup

 

Enter in the following basic setup information related to the buyer:

  1. Enter in a password for the buyer. This is case sensitive and can use a combination of letters and numbers, but no spaces. Note: the password will not be displayed when the cursor is not positioned in this field.
  2. Select if this buyer should be setup as the eCommerce Administrator. This allows for the buyer to perform a remote query of the eCommerce Server Status. This operator would normally be the DBD eCommerce Admin in your office.
  3. Select how long an operator can have a session sitting as inactive before the system will disconnect the session.
  4. Assigned to DB Distributor - Do not place anything in this field unless instructed to do so by a DB Representative.
  5. Enter in, or lookup and select, the customer code that the buyer belongs to.
  6. Location code - if you enter in a customer location here, then all orders for this buyer will be sent to this location by default. This would be known as a "location based buyer". If you leave this field blank, then the buyer will be able to choose from a drop-down list where the order should ship to. This would be known as a "corporate buyer".
  7. The supply warehouse field will display the location where the buyer will pull inventory from. This can come from the customer record, ship-to record, or the warehouse group setup using the E-Logistics module.
  8. Webcode - This field allows you to set up special pages for certain buyers if needed. The Buyer's Webcode field allows you to specify a prefix to be used for web pages for this buyer. eCommerce will attempt to locate the web pages prefixed with this entry but if not found, then the standard web page templates will be used.
  9. Entering in a vendor code will tell the system that this buyer is actually a vendor that is logging in to enter in quote information related to the RFQ module. This is not to be used with your regular buyers.
  10. Designate if this buyer is used with the Warehouse Management System.
