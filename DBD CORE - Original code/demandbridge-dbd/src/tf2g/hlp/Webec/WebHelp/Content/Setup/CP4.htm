Customer Parameters - Tab 4 - Warehouse/Catalog

 

Enter in the following information about warehouse and catalog defaults for the customer:

  1. If using the E-Logistics module, you will have access to enter in a warehouse group code. This allows you to designate a group of warehouse codes to pull inventory from, instead of just specifying a single warehouse.
  2. If a warehouse group code is not selected in step 1, you have the option of choosing if you would like the "opposite" warehouse to be looked in if there is not enough available inventory in the default warehouse. This would mean if the default warehouse of 0001, for example, was empty, the system would also look in warehouse 9001 for inventory.
  3. Designate up to five customer based catalogs. The list of available catalogs will be displayed on the right. Each buyer setup for this customer will automatically inherit the catalogs listed here.
