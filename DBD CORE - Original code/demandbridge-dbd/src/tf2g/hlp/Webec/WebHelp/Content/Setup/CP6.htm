Customer Parameters - Tab 6 - Orders

 

Enter in the following order approval information for the customer:

  1. Designate if order approval controls should be activated for this customer. Leave this blank for no approvals. Set this to an "X" to require approval for all orders. Set this to a "check" to select individual order approval situations.
  2. All Variable Print Orders - If the question is step 1 is set to an "X", this will be grayed out, as it will automatically be set to "yes". If the question in step 1 is set to a "check", you will have the option of checking this "on", in order to have order approvals turned on for all variable print orders. If the question in step 1 is set to "blank", this will be grayed out, as approvals are not used.
  3. All Over Maximum Orders - If the question is step 1 is set to an "X", this will be grayed out, as it will automatically be set to "yes". If the question in step 1 is set to a "check", you will have the option of checking this "on", in order to have order approvals turned on for all orders that are over their maximum. If the question in step 1 is set to "blank", this will be grayed out, as approvals are not used.
  4. Buyer e-mails to all? - Check this "on" to have all e-mail addresses setup under the buyer record receive approval/rejection e-mails. Having this turned off means that approval/rejection e-mails will go strictly to the buyer e-mail, not the additional e-mail addresses. You will only be allowed to set this option if step 1 is set to a "check" or an "X".
