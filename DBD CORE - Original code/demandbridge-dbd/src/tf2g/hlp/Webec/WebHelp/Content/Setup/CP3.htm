Customer Parameters - Tab 3 - Order Status

 

Enter in the following information in regards to the sorting of orders:

  1. Under the customer section, select up to three sort codes to be used for this customer. The available codes are shown on the right. Note: Codes selected in this section should be prefaced with "OS_". If your web pages support direct access of order history based upon the customer po number, then one of these codes must be based upon the customer PO number.
  2. Under the llocation section, select up to three sort codes to be used for location based buyers of this customer. The available codes are shown on the right. Note: Codes selected in this section should be based on the ship-to location address information.
