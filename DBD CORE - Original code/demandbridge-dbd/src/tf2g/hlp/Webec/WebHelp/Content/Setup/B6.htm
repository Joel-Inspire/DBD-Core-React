Buyer Setup - Tab 6 - Credit Card

 

Enter in the following information related to images for the customer:

  1. Enter in the file name for the customer's logo. Note: This will be placed on any pages where the ?tf?cust_logo? tag is used. If the customer's logo is not available, we would recommend using an image that states "your logo here". If this field is left blank, do not use the ?tf?cust_logo? tag on any pages.
  2. Enter in the file name for the company logo. Note: This will be placed on any pages where the ?tf?comp_logo? or ?tf?comp_logo_image_source? data tags are used. This field can be left blank if the default is the desired logo.
  3. Enter in the customer image path. This is the path to the directory where the customer images are stored on the server. Note: each eCommerce client should have their own directory. It is recommended that each client folder be named after the client for consistency.
  4. Optionally, if you would like to enter in a Base Page Reference to override the system default, enter it here.
  5. Optionally, if you would like to enter in a CGI Command to override the system default, enter it here.
