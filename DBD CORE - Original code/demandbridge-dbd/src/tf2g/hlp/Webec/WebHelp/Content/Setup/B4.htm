Buyer Setup - Tab 4 - Catalog

 

Enter in the following information related to catalogs for the buyer:

  1. Select if you would like for the prices to be hidden on the catalog page.
  2. For location-based buyer only, select if their catalog should only contain active items (items that have been shipped to their location before).
  3. Enter in a default catalog ID for the buyer to be associated with.
  4. If the "restrict item" option in step 2 is set to "yes", you will have the option of entering an Active Item Category. By entering a category here, the category will become the default category on the product catalog for the buyer.
  5. The secure codes access field can contain up to ten 2-digit secure codes entered in side by side. The buyer is required have the appropriate secure code if they are allowed to have access to items that have secure codes assigned to them.
  6. If you would like for PO's with this buyer to use a specific Customer PO number, you can enter this in the Default PO field.
  7. If you would like for this buyer to have access to all items, and ignore secure codes, check on the option to "ignore secure codes".
  8. Checking the "Use image view for catalog" field "on" will result in thumbnail images being shown for the items on the product catalog page, instead of just viewing rows of items without the image.
  9. You can assign up to 9 additional catalogs that the buyer will have access to. Catalog 1 on the list is always the same as the default catalog setup for them in step 3.
