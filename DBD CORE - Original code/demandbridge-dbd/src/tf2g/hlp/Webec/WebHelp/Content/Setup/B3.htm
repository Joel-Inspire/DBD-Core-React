Buyer Setup - Tab 3 - Order Status

 

Enter in the following information related to order status for the buyer:

  1. Select which order types should be displayed on the order status summary screen. The order types available are S (sales orders), R (releases), and W (billable releases).
  2. Enter in a standard sort code. The entry in this field identifies which of the user defined order status sorts, from the customer parameters, is the default sequence to be used when the buyer selects an order status link on the website. You will only be allowed to assign a sort that has been set up on the corresponding customer’s parameters . The standard available options are H1, H2, or H3 for customer based sorts, and L1, L2, or L3 for location based sorts.
  3. For location based buyers only, designate if they should only be allowed to see orders for their assigned location.
  4. Ship-to Drop Down List Group - For corporate buyers, you can designate a cost center code, which must be on a ship-to code in order for the location to show up in their drop-down list.
  5. For corporate buyers, you can designate a ship-to location that should be the defaulted location on their shipping page.
