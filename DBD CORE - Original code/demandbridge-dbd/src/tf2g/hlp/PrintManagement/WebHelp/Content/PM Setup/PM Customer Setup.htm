Item Management Customer Settings

 

DB Distributor > Item Management > Setup > Other > Setups and Parameters > Customer Parameters (FMS S2) or

DB Distributor > Accounts Receivable > Setup > Customers > Customer Default Data (A/R S1 #10)

 

There are two areas where you can setup a customer to be an Item Management customer. When setting up a new customer, or modifying an existing customer, you can flag them as an FMS customer in the Customer Default Data area of the main Customer Setup in the Accounts Receivable module. The other area to set them up is in the Customer Parameters area of the Item Management module.

 

Follow these steps to complete this panel:

  1. Enter the customer code that you want to add as an IM Customer, or edit existing IM settings for. If you type in a customer that is not currently an IM customer, it will ask you if you would like to change the customer setting to be IMS. If you are marking the customer as an IM customer in the customer defaults, you will be taken to the IM Customer Settings screen, described in step 2.
  2. The Item Management settings for the customer can be divided into the following four categories. Click on the links to see the settings for each section.
  • Processing Controls (GUI - Tab 1, CHUI - Screen I)
  • Reporting Controls (GUI - Tab 2, CHUI - Screen II)
  • Reorder Controls (GUI - Tab 2, CHUI - Screen II)
  • Item Master Setup Defaults (GUI - Tab 3, CHUI - Screen III)
  1. You will then need to setup the customersFiscal Year.
  2. Finally, Ship-to Locations will be added for the IM Customer.
