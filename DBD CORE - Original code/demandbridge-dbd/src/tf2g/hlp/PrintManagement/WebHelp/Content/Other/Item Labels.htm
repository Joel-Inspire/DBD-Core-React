Item Labels

 

DB Distributor > Item Management > Cycle Processing > Item Labels (F/M P8)

 

Similar to the Customer Mailing Labels in Accounts Receivable, the Item Labels panel is used for creating labels for your items.

 

Follow these steps to complete this panel: 

  1. Enter a range of customers and/or item codes to limit your selection.
  2. Enter the label codes to print.
  3. Enter the various information about positioning, spacing, lines, etc.
  4. Select if you want to print only IM items or all items.
  5. Review the Output Options.
  6. Click Print Report.
