IM Customer - Reorder Controls

 

Field Description
Primary Customer Contact Code This should be set as your primary contact with this customer for matters related to Print Management.
Reorder Notice Format This must be set to one of your valid reorder notice formats in order for reorder notices to be printed.
Subtract committed on RON Should committed items be subtracted from the on-hand amount when printing reorder notices.
Graph Type for RON Should the graph on the reorder notice display the current years usage, or the usage of the previous 12 months.
Reorder Notice Message If needed, you can enter in a standard message to display on reorder notices.
